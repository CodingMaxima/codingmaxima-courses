Spring Boot Properties

Property Description
server.port The port on which the server listens for incoming requests.
server.servlet.context-path The context path of the web application.
spring.application.name The name of the Spring Boot application.
spring.profiles.active The active Spring profile(s) for the application.
spring.datasource.url The URL of the database to which the application connects.
spring.datasource.username The username used to connect to the database.
spring.datasource.password The password used to connect to the database.
spring.jpa.show-sql Whether or not to output SQL statements to the console.
spring.jpa.hibernate.ddl-auto The Hibernate DDL mode used for database schema generation.
spring.jpa.properties.hibernate.dialect The Hibernate dialect used to interact with the database.
logging.level The logging level for the application.
spring.mail.host The SMTP host for sending emails.
spring.mail.port The SMTP port for sending emails.
spring.mail.username The username used for sending emails.
spring.mail.password The password used for sending emails.
spring.mail.properties.mail.smtp.auth Whether or not to use SMTP authentication for sending emails.
spring.security.user.name The username for the default Spring Security user.
spring.security.user.password The password for the default Spring Security user.
spring.thymeleaf.cache Whether or not to enable Thymeleaf template caching.
