User Class

create an constraints table of User by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. user_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. user_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. email - It should be unique and not empty (hint: use "NOT NULL CONSTRAINT and "UNIQUE CONSTRAINT")

Movie Class

create an constraints table of Movie by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. movie_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. movie_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. genre - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  4. releaseYear - It should be a four digit number (hint: use "CHECK CONSTRAINT")
  5. rating - It should be less than or equal to 5 (hint: use "CHECK CONSTRAINT")
