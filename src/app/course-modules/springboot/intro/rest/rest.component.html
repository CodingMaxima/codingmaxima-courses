Method Description
GET
  • By The HTTP GET method is used to
    Read or Retrieve
    a representation of a resource.
    This method does not require any Request Body.
    • POST
  • The POST Method is most often utilized to create new resources.
    This method contains the Request Body. POST is neither Safe nor Idempotent.
    • PUT
  • PUT request is used to Update the Existing resource.
    This method contains a Request Body PUT can also be used to
    Create a resource in the case where the resource ID is chosen by the client instead of by the server.
    • DELETE
  • Delete request is used to delete the existing resource.
    This method does not contain any Request Body.
    • PATCH
  • It is used to modify capabilities. The PATCH request only needs to contain the changes to the resource, not the complete resource.
