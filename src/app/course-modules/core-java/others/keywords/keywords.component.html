Keywords Introductions

Static Keyword
  • Memory allocation for static data happens along with the class at compile time
  • Non static data cannot be accessed from static methods. But they can be accessed with the help of objects.
  • Static data can be accessed with the help of class name which is outside the class.
  • Static block will be executed even before the execution of constructor.

    • Final Keyword
  • Once the data is declared as final that cannot be modified further.
  • When it is used before the objects, variables they can't be changed.
  • When it is used before the methods they can't be overrided.
  • When it is used before the class it can't be extended further.


    • this Keyword
  • this keyword is used to represent the current class objects.
  • it is used to differentiate the class variables and method variables when their names are same.

    • Super Keyword
  • super keyword is used to represent the parent class objects(methods and variables).
  • It can be used to call the parent class constructor, Access the parent class public and protected data.
