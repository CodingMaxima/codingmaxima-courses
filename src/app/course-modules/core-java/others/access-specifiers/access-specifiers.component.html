Keywords Introductions

Public Specifier
  • This was the most commonly used specifier.
  • Public specifer are visible in same package as well as other packages
  • They can be visible for same class and different classes(i.e., class A can be visible to class B)
    • Private Specifier
  • Private specifiers are visible in the same class only and not for the other classes in same packages as well as classes in the different packages.
    • Protected Specifier
  • Protected specifers are visible for one class to other class which is present inside the same package
  • They are invisible for one class to other class which are present in other packages. for accessing(visibility) purpose use extend
    • Default Specifier
  • The methods and variables declaredin a classs wihout any access specifers then they are called as default specifiers
  • They are visible for one class to other class which are inside the package
  • They are not visible to the classes which are present outside the packages
