Create Student class with following properties
Create a tester class and create & print atleast 5 student objects in the tester class main method.

  1. id - This property is to give unique id for student like rollnumber(int type)
  2. name - The name of the student.(string type)
  3. branch - The branch or course of the student like(cse, eee, bse, mpc and etc)(String type)
  4. section - The section of the student.(string type)
  5. totalMarks - The total marks of the student obtainded.(double type)
  6. grade - The Grade of the student(A,B,C,D and etc).(char type)
