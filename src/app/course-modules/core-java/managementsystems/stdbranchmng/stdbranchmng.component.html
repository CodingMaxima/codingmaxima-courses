Student-Branch(one-one relation)

Problem Statement Program

Student - Class
  • Integer - ID
  • String - name,
  • Branch - object
  • With constructors, all getters and setters and toString methods
    • Branch - Class
  • Integer - branch ID
  • String - branch name,
  • Integer - maximum capacity
  • With constructors, all getters and setters and toString methods
    • public Student getStudent(int id)
  • A getStudent method which accepts an integer “id” as a parameter.
  • Returns a student object with a matching ID if it exists in the students arrayList.
  • Returns “null” if no student object is present with given id in the arrayList.
    • pubic boolean addStudent(Student s)
  • An addStudent method which will let us add students into the student list.
  • This method accepts a student object, and adds into the other list based on the id.
  • This method returns a Boolean true or false, based on whether or not the student object is added in the list.
  • If we are trying to add student object with ID(4) but the arrayList already has a student object with the same ID(4) then the method must not replace the student object, but instead returns false.
    • pubic boolean deleteStudent(int id)
  • A deleteStudent method accepts a student ID as a parameter and attempts to delete a student object with the matching ID from the arrayList.
  • This method similar to addStudent also returns a Boolean value true upon successful deletion.
  • If there is no matching student object found in the arrayList, this method returns false.
    • pubic List<Student> getStudentsByBranch(Branch branch)
  • Another variation of the getStudent method is the getStudentsByBranch method.
  • Which accepts a branch object and returns a list of all the students which belong to that branch.
    • pubic void replaceStudent(Student s1, Student s2)
  • Another method present in this class is called replaceStudent.
  • This method accepts two student objects s1 and s2. We attempt to delete student s1 from the list and add student s2 into the list.
  • This method does not return anything.
    • pubic List<Student> getAllStudents()
  • The third variation of the getStudents method is the getAllStudents method.
  • Which returns the list of all students present in the arrayList.
    • public Branch getBranch(int id)
  • A getBranch method which accept an an integer ID as parameter.
  • Then returns a branch object with a matching ID.
  • If there exists no branch object with the given ID, then this method returns null.
    • pubic boolean addBranch(Branch b)
  • And addBranch method Lexus add a branch object into the branch arraylist and returns True if the addition was successful
  • If there already exists a branch with the same name, the addition is unsuccessful and the method returns false.
    • pubic boolean deleteBranch(int id)
  • A deleteBranch method deletes a branch from the list based on a given ID and returns true upon a successful deletion.
  • This method returns false if there exists no branch with the given ID in the list.
    • pubic void increaseCapacity(int capacity, Branch branch)
  • Another void method in the BranchHelper class is increase capacity which has two parametes the new capacity and branch.
  • This method will increase the capacity of the given branch.
    • pubic List<Student> getAllBranches()
  • Another variation of the getBranch method is the getAllBranches method.
  • Which returns the list of all branches present in the arrayList.
