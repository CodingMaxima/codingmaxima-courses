    • public Employee getEmployeeById(int id)
  • The getEmployeeById is a method which accepts an id of type integer as a parameter.
  • This method returns Employee object if there exists a matching id in the employees ArrayList.
  • This method returns “null” if there is no Employee object with the given id in the employees ArrayList.
    • public boolean addEmployee(Employee employee)
  • The addEmployee is a method which will let us add new employee into the employees list.
  • This method accepts an Employee object and adds it to the employees list if there is no object with same id.
  • This method returns either true or false which is of type boolean based on the situation i.e., whether the Emloyee object is added to the employees list or not.
  • For example, if we are trying to add an Employee object with id = 105 but the employee list already consist an Employee object with the same id, then the method will not add the object to the employees list instead it returns false.
    • public List<Employee> getEmployeesByDesignation(int designationId)
  • getEmployeesByDesignation is a method which accepts designationId of type integer as a parameter
  • A temporary list named desigList is created and the object in the employees list which contains the designation id equal to designationid will be added to the temporary list.
  • This method returns a all the employees addes to the temporary list i.e.,desigList.
    • public Employee updateEmployeeSalary(int id, double newSalary)
  • The updateEmployeeSalary is a method which accepts an id of type integer and newSalary of type double as parameters.
  • This method will update the current salary of the particular employee containing matching id with new salary.
  • This method will return the Employee object if the salary of the employee with particular id is updated otherwise it will return “null”.
    • public Employee updateEmployeeDesignation(int id, Designation newDesignation)
  • The updateEmployeeDesignation is a method which accepts an id and Designation object as parameters.
  • This method will search for the the employee object with matching id and updates the current designation with newDesignation.
  • This method will return the Employee object if the designation is updated otherwise it will return “null”.
    • public List<Employee> getAllEmployees()
  • The getAllEmployees is a method which does not have any parameters.
  • This method returns the list of all the employees present in the employees ArrayList.

    • public boolean removeDesignation(Designation designation)
  • The removeDesignation is a method which will let us to delete the existing designation in the designations list.
  • This method accepts a Designation object and find whether the object is present in the designations list or not. If the designation exists then it will be deleted.
  • This method returns either true or false which is of type boolean based on the situation i.e., whether the Designation object is deleted from the designations list or not.
  • For example, if we are trying to delete a designation object with id = 5 but the designations list does not consists the object with the same id, then the method will not be able to delete and it returns false.
