Daemon Thread
  • It is a thread is executing at the background.
  • Ex: Garbage collector, signal dispatcher, attach listner etc..
  • The main objective of this thread is provide support to main thread. It the main thread runs with low memory then JVM runs the Garbage collecotr to destroy the useless memory. So that it will free the memory and provides the space.
  • For checking the nature of the thread by using isDeamon().
  • syntax: public boolean isDeamon()
  • For changing the nature of a thread by using setDaemon()
  • syntax: public void setDaemon(boolean b)
  • But changing the nature deamon is possible before starting the thread only. After the thread was started if we try to change it will gives the run time exception.
