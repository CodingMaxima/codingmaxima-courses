Check given String is Palindrome or Not

Problem Statement Program Animation Video Test Cases Practice 

Palindrome String : A string is said to be palindrome if the reverse of the String is equal to the actual string.(or)
if the String left to right is equal to the string right to left.

Example:
If we have String "noon" the reverse of the given string is "noon" , the reverse of the string is equal to actual string then output should be "noon is a palindrome".

sample 1:
String = "level"
output : level is a palindrome

Sample 2 :
String = "hello"
output " hello is not a paindrome
