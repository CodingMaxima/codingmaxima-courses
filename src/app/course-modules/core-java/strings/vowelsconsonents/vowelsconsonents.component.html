Number of Vowels and consonants Count

Problem Statement Program Animation Video Test Cases Practice 

The Program is to find number of vowels and number of consonants in a String:
Except the alphabets a,e,i,o,u  remaining all the alphabets are called as consonants.

Example:
If we have a String "hello" then the output should be "string hello has 2 vowels , 3 consonants"

Sample 1 :
String = "computer"
Output :  No of vowels in the string is 3.
  No of consonants in the string is 5.

sample 2 :
String = "rythms"
Output : No of vowels in the string is o.
 No of consonants in the string is 6.

No of vowels in the string is 2.
