Reverse of a given String

Problem Statement Program Animation Video Test Cases Practice 

the program is to Print the reverse of a given String.

Example :
If we have String "java" the reverse of the given string is "avaj" , then output should be "reverse of the string java is avaj".

Sample 1 :
String = "hello"
output : reverse of the string hello is olleh

sample 2 :
String = "program"
output : reverse of the string program is margorp
