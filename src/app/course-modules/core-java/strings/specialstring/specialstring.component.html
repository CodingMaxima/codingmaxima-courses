Special String

Problem Statement Program Animation Video Test Cases Practice 

The program is to find whether the given String is an special String or not.

Special String : In a String if any two vowels were in side by side then that String is a Special String.

Example : 

If we have a String "Cream" in the given String 'e' and 'a' are vowels, 'e' is present in 3rd position and 'a' is present in 4th position. so they were present in side by side then output should be "Special String"

Sample 1 :
string = "heep"
Output : Special String.

Sample 2 :
String = "stack"
Output : "Not a Special String".
Example - 1
Input

String = {{ '{' }}"heep"{{ '}' }}

Output

heep : is a special string.

Example - 2
Input

String = {{ '{' }}"stack"{{ '}' }}

Output

stack : is not a special string.

Example - 3
Input

String = {{ '{' }}"search"{{ '}' }}

Output

search : is a special string.
