Strings Introduction

String String Buffer String Builder
Example-1

String s = "hello"

Every time whenever we are creating the string literal the JVM checks the String Pool Memory/constant(SPC). If the string is not present it will store that data value in SPC.Only one object will be created

Example-2

String s=new String("Welcome");

Whenever we are creating the string by using new keyword it will store the data value at 2 places one is in heap memory and another one is SPC

METHODS
  • concat() method - It adds an string to the end of another string.Its return type is char
  • charAt() method - It returns the character at the specified position.Its return type is string
  • contains() method - It will check whether the given string contains the sequence of characters or not.Its return type is boolean
  • equals() method - It checks the equality of string objects. It is meant for content comparision.It default considers content also.Its return type is boolean
  • equalsIgnoreCase() method - It is same as like equals method. The difference is it ignores case considerations.Its return type is boolean
  • isEmpty() method - It is used to check whether the string is empty or not.Its return type is boolean
  • length() method -It is used for checking the length of the string.Its return type is int
  • replace() method -It is used when the specified value in string have to to replaced with new string value.Its return type is string
  • split() method -It is used to split the string into an array of substrings.Its return type is string[]
  • substring() method -It is used for two types(one for begining, another is by settint the start and end range).Its return type is string
  • indexOf() method -It returns the index of specified characters from the starting.Its return type is string
  • lastIndexOf() method -It returns the index of specified characters from the ending position.Its return type is string
  • tolowercase() method -It converts the entire string into lower case letters.Its return type is string
  • touppercase() method -It converts the entire string into upper case letters.Its return type is string
  • tostring() method -It is used for returning the values of string objects.Its return type is string
  • trim() method -It removes blankspace from starting and ending ends of a string but not he middle spaces in the program.Its return type is string
