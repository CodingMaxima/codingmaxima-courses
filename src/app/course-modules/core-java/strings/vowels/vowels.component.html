Problem Statement Program Animation Video Test Cases Practice 

the Program is to find number of vowels in a String:
The alphabets a,e,i,o,u are called as vowels.

Example:
If we have String "hello" in the given string vowel 'e' is present in 2nd position and vowel 'o' is present in 5th position then we have 2 vowels in the string "hello" , then output should be "string hello has 2 vowels"

Sample 1 :
String = "computer"
Output : No of vowels in a string is 3

sample 2 :
String = "rythms"
Output : No of vowels in a string is 0
