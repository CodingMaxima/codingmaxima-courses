District Class

Create District class with following properties
Create a tester class and create & print atleast 5 district objects in the tester class main method.

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.

Here we are going to use city class object in the district class as a parameter.

Each district belongs to different cities so one city can have multiple cities.

  1. id - This property is to give unique id for the district like districtcode(string type)
  2. name - The name of the districtcode.(string type)
  3. cities - The cities present in the district.(List<City> cities - list object with City type)
  4. state - The state of the district.(string type)
