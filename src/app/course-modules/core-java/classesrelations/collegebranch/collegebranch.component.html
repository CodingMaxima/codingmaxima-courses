College Class

Create College class with following properties
Create a tester class and create & print atleast 5 college objects in the tester class main method.

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.

Here we are going to use branch class object in the college class as a parameter.

Each college belongs to different branches so one college can have multiple branches.

  1. id - The code of the college(string type)
  2. name - The name of the college.(string type)
  3. isGround - The college has ground or not(boolean type)
  4. isLibrary - The college has library or not(boolean type)
  5. isCanteen - The college has canteen or not(boolean type)
  6. branches - The list of branches in the college(List<Branch> branches - list object with Branch type)
