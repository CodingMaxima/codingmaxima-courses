Here the problem is to print the n even numbers using loops.

  • From the above given problem we have to print the n even numbers using the loops.
  • Here Is the explanation of the problem using the for loop
  • lets take n value as 3. Here the n value represents the number of even numbers which we want to print.
  • for tracking how many even numbers which we required have be printed lets take the count variable initial it with 0.
  • Step-1: Lets iterate the for loop with the initializer value as 1 because the numbers starts from one, by taking the condition as count variable is < n variable because every time count variable will increase whenever the condition becomes true until it reaches equal with the n variable and in each iteration i value will be incremented by 1
  • Step-2: Inside the for loop if condition has to be written (i % 2 == 0). by this condition if i value is divisible by 2 and gives the remainder 0 then the condition becomes true. and increases the count
  • Step-3: now iterate the loop i value is 1 and count value is 0 the n value is 3. 0 is less than 3 the condition was true . now the condition enters into the next condition 1 % 2 == 0 . this condition fails so loop will stop iteration. now i value will increased to 2
  • Step-4: Now the i value is 2 , count is 0, n value is 3. 0 is less than 3 the condition was true . Now it enters into the condition 2 % 2 == 0 . This condition was true the count value beomes 1.now i value becomes 3
  • Step-5: Now the i value is 3, count is 1, n value is 3. 1 is less than 3 the condition was true. Now it enters into the next conditon 3 % 2 == 0. This condition was false the count value remains same as 1. now i value becomes 4.
  • Step-6: Now the i value is 4 , count is 1, n value is 3. 1 is less than 3 the condition was true. Now it enters into the next condition 4 % 2 == 0. This condition was true so the count will be increased to 2. now the i value becomes 5.
  • Step-7: Now the i value is 5 , count is 2, n value is 3. 2 is less than 3 the condition was true. Now it enters into the next condition 5 % 2 == 0. This condition was false so the count value remains same as 2. Now the i value becomes 6.
  • Step-8: Now the i value is 6, count is 2, n value is 3. 2 is less than 3 the condition was true. Now it enters into the next condition 6 % 2 == 0. This condition was true so the count will be increased to 3. now the i value becomes 7.
  • Step-9: Now the i value is 7, count is 3, n value is 3. 3 is less than 3 the condition is false so the loop will terminates and program ends here.
