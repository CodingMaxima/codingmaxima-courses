Here the problem is to print the n to m numbers using the "for" loop and "while" loop.

  • From the above given problem we have to print the numbers starting from n to m numbers using the loops.
  • Here Is the explanation of the problem using the for loop
  • Step-1: In the problem we have to give the numbers to the variables n and m. n is the starting value and m is the ending value.
    lets assume the n value as 1 and m value as 5.
  • Step-2: Now the loop will starts the iteration taking the starting value n as the initializer and checks for the condition as n i.e., 1 is <= m i.e., 5 . The condition is true so the n value will becomes 2 and the m value is 5.
  • Step-3: Now the loop will starts iteration by taking the n value as 2 and checks for the condition 2 is <= 5. The condition is true so the n value will becomes 3 and the m value is 5.
  • Step-4: Now the loop will starts iteration by taking the n value as 3 and checks for the condition 3 is <= 5. The condition is true so the n value will becomes 4 and the m value is 5.
  • Step-5: Now the loop will starts iteration by taking the n value as 4 and checks for the condition 4 is <= 5. The condition is true so the n value will becomes 5 and the m value is 5.
  • Step-6: Now the loop will starts iteration by taking the n value as 5 and checks for the condition 5 is <= 5. The condition is true so the n value will becomes 6 and the m value is 5.
  • Step-7: Now the loop will starts iteration by taking the n value as 6 and checks for the condition 6 is <= 5. The condition is false so the loop will stop iterating.
