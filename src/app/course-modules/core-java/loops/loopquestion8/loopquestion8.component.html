Here the problem is to print the n odd numbers using loops.

  • From the above given problem we have to print the n odd numbers using the loops.
  • Here Is the explanation of the problem using the for loop
  • lets take n value as 2. Here the n value represents the number of odd numbers which we want to print.
  • for tracking how many odd numbers which we required have be printed lets take the count variable initial it with 0.
  • Step-1: Lets iterate the for loop with the initializer value as 1 because the numbers starts from one, by taking the condition as count variable is < n variable because every time count variable will increase whenever the condition becomes true until it reaches equal with the n variable and in each iteration i value will be incremented by 1
  • Step-2: Inside the for loop if condition has to be written (i % 2 != 0). by this condition if i value is divisible by 2 and gives the remainder 1 then the condition becomes true. and increases the count
  • Step-3: now iterate the loop i value is 1 and count value is 0 the n value is 2. 0 is less than 2 the condition was true . now it will check for next condition 1 % 2 != 0 . this condition is true.so the count is incresed to 1. now i value will increased to 2.
  • Step-4: Now for other iteration i value is 2, count is 1, and n is 2. 1 is less than 2 the condition was true. Now it will check for the other condition 2 % 2 != 0. This condtion is false so the count remains same as 1. Now i will becomes 3.
  • Step-5: Now for other iteration i value is 3, count is 1, and n is 2. 1 is less than 2 the condition was true. Now it will check for the other condition 3 % 2 != 0. This condtion is true. So the count is increased to 2. Now i will becomes 4.
  • Step-6: Now the other iteration starts with i value as 4 , count as 2 and n as 2. 2 is less than 2 the condition is false. So the loop will terminates and program ends here.
