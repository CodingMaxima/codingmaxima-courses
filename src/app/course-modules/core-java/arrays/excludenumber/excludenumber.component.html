Exclude Number while reading Array

Problem Statement Program Animation Video Test Cases Practice 
we need to read the inputs (mention the size of the inputs). 
when we read the input as 22 then we cannot take the number 22 and again need to read the input whenever the number 22 will come.

INPUT:
1
23
45
22 --- here ask the input again. 
45
22 --- here ask the input again. 
56
78

OUTPUT:
1
23
45
45
56
78
Example - 1
Input

Array = {{ '{' }}1, 6, 14, 22, 22, 85, 22, 17{{ '}' }}

exclude : 22

Output

The elements in the array is 1 6 14 85 17

Example - 2
Input

Array = {{ '{' }}15, 22, 45, 29, 22, 43{{ '}' }}

exclude : 45

Output

The elements in the array is 15 22 29 22 43

Example - 3
Input

Array = {{ '{' }}71, 73, 22, 85, 54, 73, 34{{ '}' }}

exclude : 73

Output

The elements in the array is 71 22 85 54 34
