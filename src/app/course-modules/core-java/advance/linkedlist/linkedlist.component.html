Linked List

Introduction Problem Animation Video Practice
  • LinkedList is an collection framework which contains manyobjects which are of same time and the elements are doubly linked to one another.
  • In linnked list they follow the insertion order ,accepts the duplicates & non synchronized
  • Linked list are used as lists, stack, queue
  • Linked list is good choice for the insertion and and deletion of the data.
  • There are some methods used in the linked list
  • Syntax: LinkedList <Type> linkedList = new LinkedList<>();
  • In linked list every element is known as node and consists of three fields. previous(it will stores the address of the first element and it is null for starting element),data(It will stores the actual data), next(It will stores the ending element & it is null for the ending element)
    • Methods Explanation
    addFirst() It will add the element at the starting of list.
    addLast() It will add the element at the starting of list.
    addAll() It will add all the elements of the list.
    removeFirst() It will remove the element at the starting of the list.
    removeLast() It will remove the element at the ending of the list.
    clear() It will remove all the elements from the list.
    getFirst() It will get the element at the starting of the list.
    getLast() It will get the element at the ending of the list.
    clone() It will make an copy of the linkedlist.
    size() It will show the length of the list.
    toArray() It will converts the entire elements into an array.
    set() It will change the current element with the new element
