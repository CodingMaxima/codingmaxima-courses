  • Functional interface is an interface with only one abstract method i.e., the interface implementation will only represents one behaviour
  • It can be define with the help of @FunctionalInterface annotation
  • In short it is also called as SAM interface(Single Abstract Method)
  • It provides the users with the fundamental programming approach & it is not used alone, It should be used along with lambda expression and method references inorder to make the code for more readability, clear, straight forward.
