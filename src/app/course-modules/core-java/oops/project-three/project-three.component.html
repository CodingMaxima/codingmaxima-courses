Hierarchical Inheritance Example

introduction Program
In this example, the InterestCalculator abstract class provides a common base implementation for interest calculations. It includes fields for principal amount, number of years, and interest rate. The abstract method calculateInterest() is responsible for performing the actual interest calculation, which is then implemented by its subclasses.
The SimpleInterestCalculator class extends the InterestCalculator abstract class and overrides the calculateInterest() method to calculate simple interest. The formula used is principal * rate * years.
The CompoundInterestCalculator class also extends the InterestCalculator abstract class and overrides the calculateInterest() method to calculate compound interest. The formula used is principal * (1 + rate/100)^years - principal.
The InterestCalculatorApp class demonstrates the usage of the interest calculators. It creates instances of both SimpleInterestCalculator and CompoundInterestCalculator and calls the displayInterest() method to calculate and display the interest amount.
