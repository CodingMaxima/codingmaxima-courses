Encapsulation

Encapsulation is a mechanism of combining data and methods into the single unit.

We can achieve Encapsulation with private data with public methods.

It acts like protective wall which prevents the data from being accessed by the code outside of the wall & also it hides the data in class from other classes. so it is also called as Data Hiding

Polymorphism

Polymorphism is a mechanism which allows to perform the single task in different ways.

We can achieve polymorphism by using overloading and overriding.

There are 2 types of polymorphisms

Compile time polymorphism: It is also called as overloading/ static binding. It states that class can have multiple methods with same but with different signatures i.e, number of parameters and types of parameters & each method stays as independent.

Simple defined as same methods in same class with different signatures(parameters)

Run time polymorphism: It is also called as overriding/ dynamic binding. It states that the class and subclass can have exactly same methods .Whenever the method is called it is decided at the run time to execute which class method based on the object.

simply defined as same methods in same signatures with different classes
