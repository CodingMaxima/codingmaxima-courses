Matrix Multiplication

Problem Statement Program Animation Video Test Cases Practice

Matrix multiplication is an operation performed on two matrices to produce a third matrix. It involves multiplying the elements of the rows of the first matrix with the corresponding elements of the columns of the second matrix and summing up the products.

For example, let's consider two matrices A and B:

Matrix A
2 4
1 3
Matrix B
5 2
6 1

To perform matrix multiplication, we multiply the corresponding elements of the rows and columns:

Resultant Matrix C
(2 * 5) + (4 * 6) = 22 (2 * 2) + (4 * 1) = 10
(1 * 5) + (3 * 6) = 23 (1 * 2) + (3 * 1) = 5

So, the resultant matrix C is obtained by performing matrix multiplication on matrix A and matrix B.

Matrix multiplication can be performed between matrices of compatible sizes. Specifically, the number of columns in the first matrix should be equal to the number of rows in the second matrix.

Note: Matrix multiplication is not commutative, meaning the order of multiplication matters. In the above example, Matrix A multiplied by Matrix B is not the same as Matrix B multiplied by Matrix A.
