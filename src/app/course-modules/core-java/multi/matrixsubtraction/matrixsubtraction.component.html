Matrix Subtraction

Problem Statement Program Animation Video Test Cases Practice

Matrix subtraction is an operation performed on matrices where corresponding elements from two matrices are subtracted to form a new matrix. The matrices involved in the subtraction operation must have the same dimensions, meaning they must have the same number of rows and columns.

For example, let's consider two matrices A and B:

Matrix A
5 8 10
3 2 7
Matrix B
3 4 2
1 6 5

To perform matrix subtraction, we subtract the corresponding elements at the same positions:

Resultant Matrix C
5-3 8-4 10-2
3-1 2-6 7-5
Resultant Matrix C
2 4 8
2 -4 2

So, the resultant matrix C is obtained by subtracting the corresponding elements from matrix A and matrix B.
