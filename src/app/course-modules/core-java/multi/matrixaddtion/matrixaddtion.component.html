Matrix Addition

Problem Statement Program Animation Video Test Cases Practice

Matrix addition is an operation performed on matrices where corresponding elements from two matrices are added together to form a new matrix. The matrices involved in the addition operation must have the same dimensions, meaning they must have the same number of rows and columns.

For example, let's consider two matrices A and B:

Matrix A
1 2 3
4 5 6
Matrix B
7 8 9
10 11 12

To perform matrix addition, we add the corresponding elements at the same positions:

Resultant Matrix C
1+7 2+8 3+9
4+10 5+11 6+12
Resultant Matrix C
8 10 12
14 16 18
