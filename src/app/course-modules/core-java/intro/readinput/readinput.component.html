Reading Input from Console

Introduction Program Practice
Integer

Reading the integers values by giving the inputs externally is possible by using the nextInt() method.

Double

Reading the double values by giving the inputs externally is possible by using the nextdouble() method.

Character

The character in the string can be read internally by using the next() method followed by the charAt() method.

Word

The words in the string can be read internally by using the next() method. this method reads the values only up to the white space occurs(means once the space occurs).

Sentence

The sentence in the string can be read internally by using the nextLine() method. This method will reads the enter sentence present in the line.
