Different ways of printing Hashmap

Problem Statement Animation Video

1. Using entrySet() and a for-each loop:


                        HashMap<String, Integer> hashMap = new HashMap<>();
                        // Add elements to the HashMap...
                        
                        for (Map.Entry<String, Integer> entry : hashMap.entrySet()) {{ '{' }}
                          System.out.println(entry.getKey() + " => " + entry.getValue());
                          {{ '}' }}

2. Using keySet() and a for-each loop:


                        HashMap<String, Integer> hashMap = new HashMap<>();
                        // Add elements to the HashMap...
                        
                        for (String key : hashMap.keySet()) {{ '{' }}
                          System.out.println(key + " => " + hashMap.get(key));
                          {{ '{' }}

3. Using Java 8's forEach and lambda expression:


                        HashMap<String, Integer> hashMap = new HashMap<>();
                        // Add elements to the HashMap...
                        
                        hashMap.forEach((key, value) => System.out.println(key + " => " + value));

4. Using Java 8's Stream API:


                        HashMap<String, Integer> hashMap = new HashMap<>();
                        // Add elements to the HashMap...
                        
                        hashMap.entrySet().stream()
                          .forEach(entry => System.out.println(entry.getKey() + " => " + entry.getValue()));

5. Using Iterator and entrySet():


                        HashMap<String, Integer> hashMap = new HashMap<>();
                        // Add elements to the HashMap...
                        
                        const iterator = hashMap.entrySet().iterator();
                        while (iterator.hasNext()) {{ '{' }}
                          const entry = iterator.next();
                          System.out.println(entry.getKey() + " => " + entry.getValue());
                          {{ '}' }}
