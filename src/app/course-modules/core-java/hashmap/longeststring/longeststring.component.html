Longest String

Problem Statement Program Animation Video Test Cases Practice 

Program to find longest string  in the string array using Hashmap.

Example :

If we have an array string[] = {{ '{' }} “java”,”test”,”java”,”learning”{{ '}' }} and 
we need to print the longest string in a string array. 
In the string array we can see the string “learning” is the longest(size of the string is greater than the remaining strings). 
So, we can print the string "learning” is the Longest string in the string array.

Sample 1 :
String[] =  {{ '{' }}“coding”,”maximum”,”success”,”calculation”{{ '}' }}
output : 
Longest string in the array is calculation

Sample 2 :
String[] =  {{ '{' }}“occurrence”,”one”,”management”,”strong”{{ '}' }}
output : 
Longest string in the array is occurrence
Longest string in the array is management
