HashMap Introduction

For creating an HashMap:

HashMap<Integer,String> different = new HashMap<> ();

Here, Integers,Strings are the object which can store Integer keys,String values.

HashMap<String,String> diffstrings = new HashMap<> ();

Here, strings,strings are the object which can store both String keys and values.
