Character Occurence

Problem Statement Program Animation Video Test Cases Practice 

Program to find occurrence of each character in array using Hashmap.

Example :

If we have an array char array[] = {{ '{' }}‘p’,’s’,’y’,’s’,’y’{{ '}' }} and 
the first character in the array is ‘p’ 
then the number of occurrence of character is 1 and 
the next character is ’s’. 
So, the number of occurrence of character is 2 
In the same way, we need to print the number of occurrence of every character in an array. 

Sample 1 :
Array = {{ '{' }}‘p’,’s’,’y’,’s’,’y’{{ '}' }}
output : 
The number of occurrence of character p is 1
The number of occurrence of character s is 2
The number of occurrence of character y is 2

Sample 2 :
Array = {{ '{' }}‘c’,’h’,’o’,’c’,’h’{{ '}' }}
output : 
The number of occurrence of character c is 2
The number of occurrence of character h is 2
The number of occurrence of character o is 1
