Duplicate Characters

Problem Statement Program Animation Video Test Cases Practice 

Program to find Number of duplicate characters in a string using Hashmap.

Example :

If we have a string “Radiation” and we need to print the number of duplicate characters in the string which is repeated more than once. In the string we can see the character “a” and “i” are repeated twice. 
So, we can print the number of duplicate character in the string is 2.

Sample 1 :
String =  “Occurrence”
output : 
c occurs 3 times
r occurs 2 times
e occurs 2 times

Sample 2 :
String =  “Attitude”
output : 
t occurs 3 times
