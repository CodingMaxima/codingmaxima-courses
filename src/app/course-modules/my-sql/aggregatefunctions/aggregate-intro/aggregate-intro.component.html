Aggregate Functions

Introduction
  • Aggregate functions are used to perform calculations on multiple values and gives a single value as result.
  • In aggregate funcitons they perform tasks by ingnoring null values
  • The aggregate functions are used along with the WHERE clause
  • The aggregate functions in MYSQL are (MIN, MAX, AVG, COUNT, SUM)
  • MIN: It is used to find the minimum(lowest) values present in the table columns.
  • MAX: It is used to find the maximum(highest) values present in the table columns.
  • AVG: It is used to find the Average values present in the table columns.
  • SUM: It is used to find the sum of the values present in the table colomns.
  • COUNT: It is used to count the values present in the table rows.
