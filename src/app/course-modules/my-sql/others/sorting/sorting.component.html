Question-3

Print all the employees details such as id,first_name, last_name,company_name,salary in an order of salary in decreasing order from the entire table

Descirption:- From the employee table have to find the employees id,first_name, last_name,company_name,salary in an order of salary in decreasing order

{{col}}
{{col}}
