Anomalies Introductions

UPDATION ANOMALY

When we update some rows in the table, and if it leads to the instability of the table then this anomaly occurs.

INSERTION ANOMALY

If there is a new row inserted in the table and it creates the instability in the table then it is called the insertion anomaly.

DELETION ANOMALY

If we delete some rows from the table and if any other information or data which is required is also deleted from the database, this is called the deletion anomaly in the database.
