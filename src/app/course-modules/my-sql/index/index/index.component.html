Index

Creating the index

Syntax:

  • CREATE INDEX index
  • ON TABLE column;

    • Here in this synatx index is the name given to the index, Table is the table name of the index were it is created & Column Is the name of the column on the table which it is applied.

    Ex:-create index status_index on employee(designation);

    Removing the index

    For removing the index from the database use DROP INDEX

    Syntax: ALTER TABLE EMPLOYEE DROP INDEX index-name;

    Ex:-alter table employee drop index status_index;

    Altering the index

    Syntax:

  • ALTER Table table_name
  • add INDEX index_name(column name);

    • Ex:-alter table employee add index status_index(employee_type);

    showing the index

    For showing the indexs that are created on the table present in the table.

    Syntax: SHOW INDEXES FROM TABLE;

    EX:- SHOW INDEXES FROM employee;
