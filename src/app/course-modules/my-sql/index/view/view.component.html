View

Creating the view

Syntax:

  • CREATE VIEW view_name AS
  • SELECT column1, column2, ...
  • FROM table_name
  • WHERE condition;

    • Example Query:

  • CREATE VIEW ChemistryStudents AS
  • SELECT first_name, last_name,college_name
  • FROM student
  • WHERE branch = 'Bsc.Chemistry';

    • For viewing them select * from Chemistrystudents;

    Updating the view

    For updating the view use CREATE OR REPLACE VIEW

    Syntax:

  • CREATE OR REPLACE VIEW view_name AS
  • SELECT column1, column2, ...
  • FROM table_name
  • WHERE condition;

    • In this if the view is already created then it will update the changes or else it will create the view. If we are modifying anything in already existed view replace and create should have to be mentioned compulsoryly.

    Example Query:

  • CREATE OR REPLACE VIEW ChemistryStudents AS
  • SELECT first_name, last_name,college_name,total_marks
  • FROM student
  • WHERE branch = 'Bsc.Chemistry';

    • For viewing them select * from Chemistrystudents;

    Droping the view

    Use DROP VIEW

    Syntax: DROP VIEW view_name;

    Example:-Drop VIEW Chemistrystudents;
