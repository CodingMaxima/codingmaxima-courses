Question-1

Print all the employees who has laptops along with laptops information(inner Join)

select * from employee e inner join laptop l on e.emp_id = l.emp_id;

Question-4

{{col}}
{{col}}

{{col}}

Question-6

Print the employee details who are using expired warranty laptops along with laptop details

{{col}}
{{col}}

select * from employee e inner join laptop l on e.emp_id = l.emp_id where l.lap_warranty <= curdate();

Question-7

Print the employee details who are using "Dell, Lenovo, Hp" laptops

{{col}}
{{col}}

select * from employee e inner join laptop l on e.emp_id = l.emp_id where l.lap_name in('Dell', 'Lenovo', 'HP');

Question-8

Print all the laptop company names & the count of employees who are using them.

{{col}}
{{col}}

select l.lap_name, count(e.emp_id) from employee e inner join laptop l on e.emp_id = l.emp_id group by l.lap_name;

Question-8

Extend 1 year extra warranty for the all Dell laptops which currently covers in warranty and was used by the employees

NOTE: This query works only when the safe modes were disabbled in MYSQL WORKBENCH

update employee e inner join laptop l on e.emp_id = l.emp_id set l.lap_warranty = date_add(l.lap_warranty, interval 1 year) where l.lap_warranty > curdate() and l.name='Dell';
