What are my-sql Languages?

  • DDL - Data Definition Language
  • DML - Data Manipulation Language
  • DQL - Data Query Language
  • DCL - Data Control Language

    • Question-2

    what are DDL, DML, DQL, DCL?

  • DDL : Used for creating or modifying database objects such as Tables, Indexes and users. Ex: Create, Modify, Delete
  • DML: Used for manipulating the existing data in the database. Ex: Insert, Update
  • DQL: Used for querying the data from the database Ex: Select
  • DCL: Used to control the access to the data stored in database. Ex:Grant, Revoke

    Question-5

    What are Joins?

    A join clause is used to combine rows from two or more tables based on related column between them.

  • Inner join:-It will returns all the records matching values from both the tables.
  • Left Join:- It will returns all the records from left table and matched records from the right table.
  • Right Join:- It will returns all the records from right table and matched records from the left table.
