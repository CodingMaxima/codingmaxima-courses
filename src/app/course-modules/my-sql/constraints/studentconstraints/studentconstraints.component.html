Introduction Animation video
create an constraints table of student by using below data
  • roll_number - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  • first_name - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  • last_name - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  • gender - it should be either male or female (hint: use "CHECK CONSTRAINT")
  • age - it should be in between greater than or equal to 15 and less than or equal to 50(hint: use "CHECK CONSTRAINT")
  • total_marks - keep a fixed value for it. If you doesn't want to give an value for this particular column/for auto generating of the value without giving any manual values(hint: use "DEFAULT CONSTRAINT")
  • college_name - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  • grade - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  • branch - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  • current_semister - keep a fixed value for it. If you doesn't want to give an value for this particular column/for auto generating of the value without giving any manual values(hint: use "DEFAULT CONSTRAINT")

    • CREATE TABLE student (

    roll_number int primary key,

    first_name varchar(100) not null,

    last_name varchar(100) not null,

    gender varchar(20) CHECK(gender in ('male','female')),

    age int CHECK(age >=15 AND age <=50),

    total_marks int default(0),

    college_name varchar(100) not null,

    grade varchar(20) not null,

    branch varchar(20) not null,

    current_semister int default(1)

    );
