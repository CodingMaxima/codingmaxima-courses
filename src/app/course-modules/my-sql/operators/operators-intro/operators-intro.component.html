LIKE - BETWEEN - IN

Introduction
LIKE
  • The LIKE operator is used with WHERE clause to search for a specified pattern string in a column.
  • There are two wildcars are used with the LIKE operator:
  • The % (percentage) represents zero, one, or multiple characters string
  • The _ (underscore) represents single character

    • BETWEEN
  • The BETWEEN operator selects the values within a given range(start and end).
  • The values are numbers, text, or dates.
  • The both begin range and end range values are included.

    • IN
  • The IN operator allows you to specify multiple values in a WHERE clause.
