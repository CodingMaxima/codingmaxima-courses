Reverse of Number

Problem Statement Program Animation Video Test Cases Practice

The program is to print the given number in a reverse order.

Reverse of a number :- Exhibiting the given number in the reverse order.

Example 1:
if your program reads the value 123456 from the keyboard
it should print the value as 654321

Example 2:
if your program reads the value 123400 from the keyboard
it should print the value as 4321 (NOT 004321)

Example - 1
Input

Enter a number

123456

Output

Reverse of a given number 123456 is 654321

Example - 2
Input

Enter a number

123400

Output

Reverse of a given number 123400 is 4321

Example - 3
Input

Enter a number

17652

Output

Reverse of a given number 17652 is 25671
