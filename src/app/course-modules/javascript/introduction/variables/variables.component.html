Variables

In javascript variables are fundamental components that allow you to store, manipulate, and reference data in your programs. They serve as named containers or placeholders for values.

Variable Declaration
  • The values in the array can be by using the index which starts from 0 and continues so on....
  • To create a variable in JavaScript, you use the var, let, or const keyword, followed by a variable name.
  • Variable names are case-sensitive, must begin with a letter, underscore (_), or dollar sign ($), and can contain letters, digits, underscores, and dollar signs.
    • Assignment
  • After declaring a variable, you can assign a value to it using the assignment operator (=).
  • Variables can be assigned values of different types, including numbers, strings, booleans, objects, arrays, and more.
    • Data Types
  • JavaScript is dynamically typed, meaning variables are not bound to a specific data type. Their types are determined at runtime based on the values they hold.
  • Common data types include numbers, strings, booleans, objects, arrays, and functions.
    • Variable Scope
  • Variables in JavaScript have scope, which defines where in your code a variable is accessible.
  • Variables declared with let and const have block scope, which means they are limited to the block (inside curly braces {{ "{" }} {{ "}" }}) where they are defined.
  • Variables declared with var have function scope, making them accessible throughout the entire function where they are declared.
    • Hoisting
  • JavaScript variables are hoisted, which means they are moved to the top of their containing function or block during compilation.
  • While the variable declarations are hoisted, the assignments remain in place, so the value is undefined until assigned.
    • Constants
  • Variables declared with const are constants, meaning their values cannot be changed after the initial assignment.
  • This is useful when you want to define values that should remain constant throughout your program.
