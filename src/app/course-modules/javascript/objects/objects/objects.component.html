Objects

In JavaScript, objects are fundamental data structures that play a main role in organizing and manipulating data. They are a collection of key-value pairs, where each key (also called a property) maps to a value. They are a key part of the language's design and are used to represent and manage data in a structured and flexible manner. Objects in JavaScript are a cornerstone of its object-oriented programming (OOP) capabilities, allowing you to model real-world entities and their properties.

Object Basics
  • In JavaScript, an object is a collection of key-value pairs. The keys (also known as properties) are strings or Symbols, and the values can be of any data type, including other objects, functions, primitive data types (like numbers and strings), and even arrays.

    • Example for creating a simple object:

    const person = {{ "{" }}

      name: "John",
      age: 30

    {{ "}" }};


  • In this example, 'person' is an object with two properties: 'name' and 'age'. The 'name' property has a string value, and the 'age' property has a numeric value.
    • Accessing Object Properties

    For accessing the object properties 2 ways are there dot notation or bracket notation

  • Dot Notation Example:

    • console.log(person.name);

    console.log(person.age);


  • Bracket Notation Example

    • console.log(person['name']);

    console.log(person['age']);


  • Bracket notation is useful when the property name is dynamic or contains special characters.
    • Adding and Modifying Properties
  • You can add new properties to an object or modify existing ones at any time.

    • Example:

    person.gender = "Male";

    person.age = 31;


    console.log(person.gender);

    console.log(person.age);


  • In the above example based on the first line and second line the output will be modified and gives the modified result.
    • Deleting Properties
  • To remove a property from an object, you can use the delete operator.

    • Example code

    delete person.age;

    console.log(person.age);


  • Based on the above code by deleting the preson.age property in the result it gives undefined because there is no age property left in the object.

    Object Constructors and Classes
  • JavaScript also supports the concept of constructor functions and classes for creating objects with similar structures and behaviors. This allows you to create multiple instances of objects with the same properties and methods.

    • Constructor Function Example:

      this.name = name;
      this.age = age;

    {{ "}" }}

    const john = new Person("John", 30);

    Object Destructuring
  • You can easily extract object properties and assign them to variables using object destructuring:

    • Example Code:

    const {{ "{" }} name, age {{ "}" }} = person;

    console.log(name, age);

    Object Cloning
  • To create a copy of an object, you can use various techniques, such as the spread operator ( '{{ "{" }} ... {{ "}" }}''), Object.assign(), or the 'JSON.parse()' and 'JSON.stringify()' methods.

    • Example Code:

    const copyPerson = {{ "{" }}...person {{ "}" }};

    Object Serialization and Deserialization
  • You can convert an object to a JSON string and vice versa using JSON.stringify() and JSON.parse():

    • Example Code:

    const personJSON = JSON.stringify(person);

    const personCopy = JSON.parse(personJSON);

    Object Methods and Utility Functions

    JavaScript provides several built-in object methods and utility functions for working with objects, including:

  • Object.keys(obj): Returns an array of object keys.
  • Object.values(obj): Returns an array of object values.
  • Object.entries(obj): Returns an array of key-value pairs.
  • Object.hasOwnProperty(key): Checks if an object has a specific property.
  • Object.freeze(obj): Prevents modifications to an object.
  • Object.seal(obj): Prevents adding or deleting properties but allows property value changes.
  • Object.assign(target, ...sources): Copies properties from one or more source objects to a target object.
