Strings

Introduction Problems Animation Video

In JavaScript, a string is a sequence of characters enclosed within single ('') or double ("") quotation marks. Strings are used to represent and manipulate textual data. They can contain letters, numbers, symbols, and whitespace.

Strings in JavaScript are immutable, meaning once created, their values cannot be changed. However, string methods return new modified strings rather than modifying the original string itself.

Overall, strings play a crucial role in JavaScript for storing and manipulating textual data, and their versatility makes them widely used in web development and other JavaScript applications.

Some key points in strings
Declaration and Assignment

Strings can be declared and assigned to variables using the 'var', 'let', or 'const' keywords

Examples:

var message = "Hello, World!"; (using double quotes)

let name = 'John'; (using single quotes)

const city = `New York`; (using backticks (template literals))

String Concatenation

Strings can be concatenated (combined) using the '+' operator.

Example:

code
  • var greeting = "Hello";
  • var name = "John";
  • var message = greeting + " " + name;

    • output

    "Hello John"

    • String Methods

    JavaScript provides several built-in methods to manipulate strings.

  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.
  • indexOf(substring): Returns the index of the first occurrence of a substring.
  • substring(startIndex, endIndex): Extracts a portion of the string between the given indices.
  • split(separator): Splits the string into an array of substrings based on the specified separator.
  • replace(oldValue, newValue): Replaces occurrences of a substring with a new value.
  • trim(): Removes leading and trailing whitespace from the string.

    • Example:

    code
  • var text = " Hello, World! ";
  • var upperCaseText = text.toUpperCase();
  • var index = text.indexOf("World");
  • var trimmedText = text.trim();

    • output
  • " HELLO, WORLD! "

  • "Hello, World!"

    • String Methods

    There are some commonly used string methods in JavaScript. Each method performs a specific operation on strings, allowing you to manipulate, search, and transform string data in various ways.

    1.Length-related Methods:

    length: Returns the length of a string.

    2.Case Conversion Methods:

  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.

    • 3.Substring Extraction Methods:

  • charAt(index): Returns the character at the specified index.
  • charCodeAt(index): Returns the Unicode value of the character at the specified index.
  • substring(startIndex, endIndex): Returns a new string that contains a portion of the original string between the specified indices.
  • slice(startIndex, endIndex): Returns a new string that contains a portion of the original string between the specified indices.
  • substr(startIndex, length): Returns a new string that contains a specified number of characters from a starting index.

    • 4.Searching and Position Methods:

  • indexOf(substring, startIndex): Returns the index of the first occurrence of a substring within a string.
  • lastIndexOf(substring, startIndex): Returns the index of the last occurrence of a substring within a string.
  • startsWith(substring): Checks if the string starts with the specified substring.
  • endsWith(substring): Checks if the string ends with the specified substring.
  • includes(substring): Checks if the string contains the specified substring.

    • 5.String Modification Methods:

  • concat(string1, string2, ..., stringN): Combines two or more strings and returns a new string.
  • replace(oldValue, newValue): Replaces occurrences of a substring with a new substring.
  • trim(): Removes leading and trailing whitespace from a string.
  • trimStart(): Removes leading whitespace from a string (ES2019).
  • trimEnd(): Removes trailing whitespace from a string (ES2019).
  • padStart(targetLength, paddingString): Pads the start of a string with another string until it reaches the specified length (ES2017).
  • padEnd(targetLength, paddingString): Pads the end of a string with another string until it reaches the specified length (ES2017).

    • 6.String Splitting and Joining Methods:

  • split(separator): Splits a string into an array of substrings based on a specified separator.
  • join(separator): Joins the elements of an array into a single string using a specified separator.

    • 7.String Character Iteration Methods:

  • forEach(callbackFn): Executes a provided function once for each character in the string.
  • charAt(index): Returns the character at the specified index.
  • charCodeAt(index): Returns the Unicode value of the character at the specified index.

    • 8.Regular Expression Methods:

  • match(regexp): Searches the string for a match against a regular expression and returns an array of matches.
  • search(regexp): Searches the string for a specified value or regular expression and returns the index of the first occurrence.
  • replace(regexp, replacement): Replaces matches in the string with a specified value or the result of a function.
  • split(regexp|separator): Splits the string into an array of substrings using a specified regular expression or separator.
