Arrays

Introduction Problems Animation Video

In JavaScript, an array is a data structure that allows you to store multiple values in a single variable. It is a container that holds a fixed number of elements, and each element in the array can be accessed using an index.

Simple Example

In this example creating the array called cities that contains these elements: 'New York', 'London', 'Paris', 'Mumbai', 'Delhi'.

let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];

You can access individual elements in an array using their index. The index of the first element is 0, the second element is 1, and so on. Here's an example:

  • console.log(cities[0]);
  • console.log(cities[2]);
  • console.log(cities[4]);

    • output
  • 'New York'
  • 'Paris'
  • 'Delhi'

    • In the above example, we use the square brackets [] with the index to access specific elements in the cities array.

    Arrays have several built-in functions or methods that can be used to perform various operations on the array. Let's look at a few commonly used array functions in JavaScript:
    push

    Adds one or more elements to the end of an array.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];
  • cities.push('Tokyo');
  • console.log(cities);

    • output

    ['New York', 'London', 'Paris', 'Mumbai', 'Delhi', 'Tokyo']

    pop

    Removes the last element from an array and returns it.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi', 'Tokyo'];
  • let lastCity = cities.pop();
  • console.log(lastCity);
  • console.log(cities);

    • output
  • 'Tokyo'
  • ['New York', 'London', 'Paris', 'Mumbai', 'Delhi']
    • length

    Returns the number of elements in an array.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];
  • console.log(cities.length);

    • output

    5

    indexOf

    Returns the first index at which a given element can be found in the array, or -1 if it is not present.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];
  • console.log(cities.indexOf('London'));
  • console.log(cities.indexOf('Berlin'));

    • output
  • 1
  • -1
    • slice

    Returns a new array containing a portion of the original array, specified by the start and end indices.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];
  • let selectedCities = cities.slice(0, 3);
  • console.log(selectedCities);

    • output

    ['New York', 'London', 'Paris']

    Arrays in JavaScript are very versatile and provide many other methods for manipulation and iteration, such as forEach, map, filter, and more.
