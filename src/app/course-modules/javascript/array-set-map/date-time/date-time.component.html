Date - time

Dates and times are handled using the Date object, which is part of the core JavaScript language. The Date object provides methods and properties to work with dates, times, and time zones.

Creating Date Objects
  • You can create a Date object in several ways:

    • Example for Current Date and Time:

    const currentDate = new Date();


    Example by Using a Specific Date and Time:

    const specificDate = new Date("2023-09-21T12:00:00");


    Example by Using Timestamp (milliseconds since January 1, 1970, UTC):

    const timestampDate = new Date(1632192000000);

    // 1632192000000 is a timestamp

    Getting Date and Time Components
  • You can extract various components of a Date object using its methods:

  • const date = new Date();
  • const year = date.getFullYear();
  • const month = date.getMonth(); // Month is 0-indexed (0 = January)
  • const day = date.getDate();
  • const hours = date.getHours();
  • const minutes = date.getMinutes();
  • const seconds = date.getSeconds();
  • const milliseconds = date.getMilliseconds();
    • Setting Date and Time Components
  • You can also set individual components of a Date object:

  • const date = new Date();
  • date.setFullYear(2023);
  • date.setMonth(8); // 8 = September
  • date.setDate(21);
  • date.setHours(12);
  • date.setMinutes(0);
  • date.setSeconds(0);
  • date.setMilliseconds(0);

    Working with Timezones
  • JavaScript's Date object uses the local time zone of the user's device. To work with different time zones, you can use libraries like 'luxon', 'date-fns-tz', or 'moment-timezone'.
    • Performing Date Arithmetic
  • You can perform date arithmetic by using the Date object's methods. For example, to add or subtract days from a date:

    • Example code:

    const date = new Date();

    date.setDate(date.getDate() + 7); // Add 7 days
