Maps

Introduction Problems Animation Video

A Map is a built-in object that allows you to store key-value pairs. It provides a way to associate values (the "values") with unique keys. Unlike Sets, Maps preserve the order of elements and can use any value (primitive or object) as a key.

Creating the Map

let map = new Map();

Adding Key-Value Pairs to a Map

For adding the keys and value pairs to map use the set function to add key-value pairs to the map. Each key is unique, and the associated value can be of any type.

In this example we are creating a Map called cityPopulationMap and adding key-value pairs representing cities and their respective populations.


code
  • let cityPopulationMap = new Map();
  • cityPopulationMap.set('New York', 8623000);
  • cityPopulationMap.set('London', 8908081);
  • cityPopulationMap.set('Paris', 2140526);
  • cityPopulationMap.set('Mumbai', 18410000);
  • cityPopulationMap.set('Delhi', 30240000);

    • Checking the Size of a Map

    The size property returns the number of key-value pairs in the Map.


    code

    let cityPopulationMap = new Map(['New York', 8623000], ['London', 8908081],['Paris', 2140526],['Mumbai', 18410000],['Delhi', 30240000]);

  • console.log(cityPopulationMap.size);

    • output

    5

    • Getting the Value for a Key in a Map

    The get function is used to retrieve the value associated with a specific key in the Map. If the key is not found, it returns undefined


    code

    let cityPopulationMap = new Map(['New York', 8623000], ['London', 8908081],['Paris', 2140526],['Mumbai', 18410000],['Delhi', 30240000]);

  • console.log(cityPopulationMap.get('Paris'));
  • console.log(cityPopulationMap.get('Tokyo'));

    • output
  • 2140526

    • Removing a Key-Value Pair from a Map

    The delete function is used to remove a key-value pair from the Map. After deleting, has will return false for that key.


    code

    let cityPopulationMap = new Map(['New York', 8623000], ['London', 8908081],['Paris', 2140526],['Mumbai', 18410000],['Delhi', 30240000]);

  • cityPopulationMap.delete('Delhi');
  • console.log(cityPopulationMap.has('Delhi'));

    • output
  • false

    Maps in JavaScript provide a convenient way to store and retrieve key-value pairs, preserving the order of elements. They offer functions to add, check, retrieve, remove, and iterate through the key-value pairs. Maps are useful when you need to associate data with specific keys and perform operations based on those associations.
