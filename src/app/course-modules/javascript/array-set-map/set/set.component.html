Sets

Introduction Problems Animation Video

A Set is a built-in object that allows you to store unique values of any type. It provides a way to store collections of values without duplicates. Unlike arrays, Sets do not have indexes and the elements are not stored in a specific order.

Creating the set

let set = new Set();

Adding Elements to a Set

For adding the elements to set use the add function to add elements to the set. Each element is unique, so duplicates are automatically removed.


code
  • let cities = new Set();
  • cities.add('New York');
  • cities.add('London');
  • cities.add('Paris');
  • cities.add('Mumbai');
  • cities.add('Delhi');
  • console.log(cities);

    Checking the Size of a Set

    The size property returns the number of elements in the Set.


    code
  • let cities = (['New York', 'London', 'Paris', 'Mumbai', 'Delhi']);
  • console.log(cities.size);

    • output

    5

    Checking if an Element Exists in a Set

    The has function is used to check if a specific element exists in the Set. It returns true if the element is present and false otherwise.


    code
  • let cities = (['New York', 'London', 'Paris', 'Mumbai', 'Delhi']);
  • console.log(cities.has('London'));
  • console.log(cities.has('Berlin'));

    • output
  • true
  • false
    • Removing an Element from a Set

    The delete function is used to remove an element from the Set. After deleting, has will return false for that element.


    code
  • let cities = (['New York', 'London', 'Paris', 'Mumbai', 'Delhi']);
  • cities.delete('Delhi');
  • console.log(cities.has('Delhi'));

    • output

    false

    • Clearing a Set

    The clear function is used to remove all elements from the Set, making it empty.


    code
  • let cities = ['New York', 'London', 'Paris', 'Mumbai', 'Delhi'];
  • cities.clear();

    Sets in JavaScript provide a convenient way to store unique values and perform operations such as adding, checking, removing, and iterating through the elements. It's important to note that Sets only allow unique values, so duplicates are automatically eliminated.
