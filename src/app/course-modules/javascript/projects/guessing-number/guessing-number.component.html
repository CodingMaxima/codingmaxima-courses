Project Description
"Guess My Number!" is an interactive web-based game that challenges players to guess a randomly generated number within a given range. Created using HTML, CSS, and JavaScript, the game provides an engaging and entertaining experience for players as they attempt to match the secret number.
Features
  • Random Number Generation: The game generates a secret number between 1 and 20 (configurable) at the start of each round.
  • User Input: Players can enter their guessed number in an input field and submit it to check if it matches the secret number.
  • Message Display: The game displays dynamic messages to inform players about the outcome of their guess, whether it's too high, too low, or correct.
  • Score Tracking: The game tracks the player's score, which starts at 20 and decreases with each incorrect guess. The aim is to guess the number with the highest possible score.
  • Highscore: The game keeps track of the player's highest score (remaining attempts) achieved throughout multiple rounds.
  • Background and Number Styling: The background color and size of the secret number element change when the player wins the game, providing visual feedback.
  • Restart Option: Players can restart the game by clicking the "Again!" button, resetting the score, the secret number, and the game messages.
    • Implementation
  • HTML Structure: The HTML layout consists of a header, main content area, two sections for user input and game information, and an input field and button for submitting guesses.
  • CSS Styling: CSS styles are applied to elements to create an appealing design. The color scheme, font, and layout enhance the visual experience.

    JavaScript Logic: The JavaScript code handles the game's core functionality.

  • The secret number is generated using Math.trunc(Math.random() * 20) + 1.
  • Functions are used to display messages, update the score, and display the secret number.
  • The check function evaluates the user's guess and updates the score, message, and secret number display accordingly.
  • The again function resets the game by generating a new secret number, resetting the score, and clearing messages.

  • Dynamic Message Display: The displayMessage function changes the message displayed to the player based on the outcome of their guess.
  • Score and Number Display: The displayScore and displayNumber functions update the displayed score and secret number.
  • Button and Input Field Handling: Event listeners are added to the "Check!" and "Again!" buttons to trigger the corresponding functions when clicked.
  • Background and Number Styling: The game updates the background color and secret number size when the player wins.
  • Game Restart: The "Again!" button allows players to start a new round with a fresh score and secret number.
