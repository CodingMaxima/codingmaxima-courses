Project Description
The "Digital Clock" project is a web-based application that displays the current time in both 24-hour and 12-hour formats. Created using HTML, CSS, and JavaScript, this clock provides users with an accurate and visually appealing way to keep track of time.
Features
  • Clock Display Modes: The application offers two display modes: 24-hour and 12-hour formats. Users can toggle between these modes using buttons.
  • Dynamic Time Updates: The clock constantly updates to show the current time accurately, down to the seconds.
  • Responsive Design: The clock is designed to be responsive and adaptable, ensuring a consistent user experience across different devices and screen sizes.
  • Interactive Buttons: The buttons labeled "24 Hours" and "12 Hours" allow users to switch between the two display formats effortlessly.
  • Active State Indicator: When a format button is selected, it becomes visually distinct (highlighted) to indicate the currently active format.
  • Stylish Clock Appearance: The clock is styled using CSS to enhance its appearance. The clock's digits, along with the AM/PM indicator, are displayed in an eye-catching manner.
  • Custom Header: The app features a custom header with a title and logo, adding a professional touch to the interface.
    • Implementation
  • HTML Structure: The HTML structure outlines the layout of the app. It includes the header, format selection buttons, and the clock display area.
  • CSS Styling: CSS styles are applied to various elements, including buttons, clock display, and the custom header. The styles enhance the visual appeal of the app.
  • JavaScript Logic: JavaScript is responsible for updating the clock's time, handling the format toggling, and maintaining the active state of buttons.
  • Clock Updating: The updateTime function is used to obtain the current time, format it according to the selected mode, and update the clock display accordingly.
  • Time Format Conversion: When switching to the 12-hour format, the script adjusts the hours and adds the AM/PM indicator as appropriate.
  • Format Toggling: The show24HourFormat and show12HourFormat functions are used to switch between the two display modes. These functions also manage the active state of the format buttons.
  • Clock Initialization: The setInterval function calls the updateTime function every second to ensure the clock's accuracy.
  • Default Format: The show12HourFormat function is invoked initially to set the clock to the 12-hour format when the page loads.
