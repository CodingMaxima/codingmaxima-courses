DOM

Introduction Problems Animation Video
the Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree-like structure, with each element in the document being a node in the tree. The DOM allows JavaScript to interact with the elements and modify their content, attributes, and styles dynamically.
Dom Functions and methods
getElementById

Retrieves an element from the document based on its unique identifier.

Example

code
  • // HTML: < div id="myDiv"> Hello, World!</div >
  • var element = document.getElementById("myDiv");
  • console.log(element.innerHTML);

    • output
  • Hello, World!
    • getElementsByClassName

    Returns a collection of elements that have a specific class name. // Output:

    Example

    code
  • // HTML: < p class="highlight">This is a paragraph.</p>
  • var elements = document.getElementsByClassName("highlight");
  • console.log(elements[0].innerHTML);

    • output
  • This is a paragraph.
    • getElementsByTagName

    Returns a collection of elements with the specified tag name.

    Example

    code
  • // HTML: < p> This is the first paragraph.</p> <p>This is the second paragraph.</p>
  • var paragraphs = document.getElementsByTagName("p");
  • console.log(paragraphs[0].innerHTML);

    • output
  • This is the first paragraph.
    • querySelector

    Returns the first element that matches a specific CSS selector.

    Example

    code
  • // HTML:<pclass="highlight">This is a paragraph.</p>
  • var element = document.querySelector(".highlight");
  • console.log(element.innerHTML);

    • output
  • This is a paragraph.
    • querySelectorAll

    Returns all elements that match a specific CSS selector.

    Example

    code
  • // HTML: < p class="highlight"> This is a paragraph.</p > <p >This is another paragraph.</p >
  • var elements = document.querySelectorAll("p");
  • console.log(elements[1].innerHTML)

    • output
  • This is another paragraph.
    • createElement

    Creates a new HTML element.

    Example

    code
  • var newElement = document.createElement("div");
  • newElement.innerHTML = "New element";
  • document.body.appendChild(newElement);

    • output

    • appendChild

    Appends a child element to a parent element.

    Example

    code
  • var parentElement = document.getElementById("parent");
  • var childElement = document.createElement("div");
  • parentElement.appendChild(childElement);

    Removes a child element from its parent element.

    Example

    code
  • var parentElement = document.getElementById("parent");
  • var childElement = document.getElementById("child");
  • parentElement.removeChild(childElement);
