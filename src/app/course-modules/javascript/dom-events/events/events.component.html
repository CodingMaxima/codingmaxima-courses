Mouse Events

mouseenter: Occurs when the mouse pointer enters an element.

mouseleave: Occurs when the mouse pointer leaves an element.

mouseover: Occurs when the mouse pointer is moved over an element.

mouseout: Occurs when the mouse pointer is moved out of an element.

    Event Handling Methods
    There are multiple ways to handle events depending on the specific requirements and programming style.
    Inline Event Handlers

    Event handlers can be directly specified within HTML tags using inline event attributes.

    conclick: Specifies the code to be executed when an element is clicked.

    ondblclick: Specifies the code to be executed when an element is double-clicked.

    onmouseover: Specifies the code to be executed when the mouse pointer is moved over an element.

    onmouseout: Specifies the code to be executed when the mouse pointer is moved out of an element.

    onkeydown: Specifies the code to be executed when a key is pressed down.

    onkeyup: Specifies the code to be executed when a key is released.

    DOM Event Handlers

    Event handlers can be assigned to DOM elements using their corresponding properties.

    element.onclick: Assigns a function to be executed when the element is clicked.

    element.ondblclick: Assigns a function to be executed when the element is double-clicked.

    element.onmouseover: Assigns a function to be executed when the mouse pointer is moved over the element.

    element.onmouseout: Assigns a function to be executed when the mouse pointer is moved out of the element.

    element.onkeydown: Assigns a function to be executed when a key is pressed down.

    element.onkeyup: Assigns a function to be executed when a key is released.

    Event Listeners

    Event listeners provide a more flexible and recommended approach to handle events. They allow multiple functions to be attached to an event and support event delegation.

    element.addEventListener(event, listener): Adds an event listener to the specified element.

    element.removeEventListener(event, listener): Removes an event listener from the specified element.

    Event Handlers with named functions

    Instead of using anonymous functions, you can define named functions and use them as event handlers.

    function myFunction() {{ "{" }} ... {{ "}" }}

    element.addEventListener(event, myFunction)

    Event Object

    Event handlers receive an event object as an argument, which contains information about the event and the element it occurred on. The event object has properties like target, type, keyCode, etc.

    Event Delegation

    By attaching event listeners to a parent element, you can handle events for multiple child elements efficiently. This is useful for dynamically created or large sets of elements.
