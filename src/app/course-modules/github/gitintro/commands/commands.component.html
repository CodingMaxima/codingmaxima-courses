GIT Commands

Command Description Example
git init Initializes a new Git repository. git init
git clone Creates a copy of a remote repository on your local machine. git clone https://github.com/example/repo.git
git add Adds changes to the staging area. git add filename.txt
git commit Records changes to the repository with a commit message. git commit -m "Added a new feature"
git status Shows the status of changes (untracked, modified, or staged). git status
git pull Fetches changes from a remote repository and merges them into the current branch. git pull origin master
git push Pushes local changes to a remote repository. git push origin master
git branch Lists, creates, or deletes branches. git branch (list branches)
git branch feature-branch (create branch)
git branch -d feature-branch (delete branch)
git checkout Switches to a different branch. git checkout feature-branch
git merge Combines changes from different branches. git merge feature-branch
git log Shows a log of commits. git log
git remote Manages remote repositories. git remote add origin https://github.com/example/repo.git
git remote -v (view remotes)
git fetch Downloads objects and refs from a remote repository. git fetch origin
git fetch --all (fetch from all remotes)
git reset Resets the current branch to a specific commit. git reset --hard commit-hash
git diff Shows changes between commits, commit and working tree, etc. git diff
