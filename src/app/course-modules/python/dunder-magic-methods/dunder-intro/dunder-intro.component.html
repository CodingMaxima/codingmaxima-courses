Dunder-Magic methods Introduction

Introduction

Dunder/Magic Methods are the special methods Which contains 2 underscore symbols at the starting and the ending of the method.

"Dunder" is short for "double underscore,"

For an class these are the default methods.

In python every class which we creates is an child class for the object class. so in object class the dunder method will be present.

Whether we take/ or not the object class it was an parent class to the classes that we created.

The build-in classes in python contains many magic mthods .To see the build-in classes use dir() function then you will get all the magic methods present in python.

These are used for the operator overloading( provided additional functionality to the operators, the python indirectly calls the magic methods to provide an additional functionality to it).

Few Examples of magic methods are __iter__, __call__, __contains__, __repr__, __add__, __new__(), __init__, __len__ etc.....
