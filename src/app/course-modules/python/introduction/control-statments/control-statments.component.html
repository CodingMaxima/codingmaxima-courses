Control Flow Statments

    Control Flow Statements in Python are programming constructs that determine the sequence in which the code is executed.
    They allow the programmer to control the flow of the program based on conditions, loops, and decision-making structures.

  • The if statement in Python is used to perform a certain action or execute a block of code if a specified condition is evaluated as True.
  • It provides a way to make decisions within the code based on the result of the condition.

    • Syntax of the if Statement:

      if condition:
        # Code block to be executed if the condition is True
        # This code is indented and runs when the condition is met

    Example:

      x = 10
      if x > 0:
        print("x is positive")

    From the above example Since the value of x is not greater than 10, therefore the line 'y=25' and 'print("greater than 10")' don't get executed. But the line 'print(f"End of the program with {{'{' }}y={{ '}'}}")' is executed, since it is out of the indent and is no longer based on the condition

  • The if-else statement in Python is used to create a decision-making structure where one block of code is executed if a specified condition is True, and another block of code is executed if the condition is False.
  • This statement provides an alternative execution path based on whether the condition evaluates to True or False.

    • Syntax of the if-else Statement:

      if condition:
        # Code block to be executed if the condition is True
        # This code is indented and runs when the condition is met
      else:
        # Code block to execute if the condition is False
        # This code is executed when the condition is not met

    Example:

      x = 10
      if x % 2 == 0:
        print("x is even")
      else:
        print("x is odd")

    From the above example since the value of x is not greater than 10, therefore the line 'y=25' and 'print("greater than 10")' don't get executed, but since we have an 'else' statement, the lines 'y=30' and 'print("less than 10")' are executed instead. Followed by the line 'print(f"End of the program with {{'{' }}y={{ '}'}}")' is executed, since it is out of the indent

  • The if-elif-else statement in Python allows you to evaluate multiple conditions sequentially.
  • It provides a way to check several conditions one by one and execute specific blocks of code based on the first condition that evaluates to True.
  • If none of the conditions are True, the code block under the else statement (if provided) will be executed.

    • Syntax of the if-elif-else Statement:

      if condition:
        # Code block to be executed if the condition is True
        # This code is indented and runs when the condition is met
      elif condition2:
        # Code block to execute if condition1 is False and condition2 is True
        # This code runs when condition2 is met
      elif condition3:
        # Code block to execute if conditions 1 and 2 are False, and condition3 is True
        # This code runs when condition3 is met
      # ... (additional elif blocks if needed)
      else:
        # Code block to execute if none of the above conditions are True
        # This code runs when no previous condition is met

    Example:

      score = 75
      if score >= 90:
        print("Grade: A")
      elif score >= 80:
        print("Grade: B")
      elif score >= 70:
        print("Grade: C")
      elif score >= 60:
        print("Grade: D")
      else
        print("Grade: F")

    from the above example Since the value of x is not greater than 10, therefore the line 'y=25' and 'print("greater than 10")' don't get executed, but since x is greater than 5 and we have an 'elif' statement that says x>5, the lines `y=125` and 'print("greater than 5 but less than 10")' are executed instead. Followed by the line 'print(f"End of the program with {{'{'}}y={{'}'}}")' is executed, since it is out of the indent Note: There can be any number of elifs in an if-elif-else conditional structure

  • For loop is used to iterate over a sequence of elements, such as lists, tuples, strings, dictionaries, or ranges.
  • It executes a block of code for each element in the sequence.

    • Syntax of the For Loop:

      for item in sequence:
        # Code block to be executed for each iteration

    Example 1: Iterating Through a List

      fruits = ["apple", "banana", "orange"]
      for fruit in fruits:
        print(fruit)

    Example 2: Using range() Function

      for i in range(5):
        print(i)

  • While loop is used to repeatedly execute a block of code as long as a specified condition remains True.
  • It keeps iterating until the condition becomes False.

    • Syntax of the While Loop:

      while condition:
        # Code block to be executed as long as the condition is True

    Example

      count = 0
      while count < 5:
        print(count)
        count += 1

  • Break statement is used to terminate or exit a loop prematurely based on a certain condition.
  • When encountered within a loop, it immediately stops the execution of the loop and transfers control to the next statement after the loop.

    • Syntax of the break Statement:

      for item in sequence:
        if condition:
          break
        # Code block

    Example:

      for i in range(5):
        if i == 3:
          break
        print(i)

  • Continue statement is used within loops to skip the remaining code inside the loop for the current iteration and move to the next iteration.
  • When encountered, it immediately causes the loop to go to the next iteration, skipping any code that follows the continue statement within that iteration.

    • Syntax of the break Statement:

      for item in sequence:
        if condition:
          continue
        # Code block

    Example:

      for i in range(5):
        if i == 2:
          continue
        print(i)
