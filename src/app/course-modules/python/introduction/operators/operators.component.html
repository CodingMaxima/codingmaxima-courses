Operators

Operators in Python are special symbols or keywords used to perform specific operations on one or more values (operands). They enable various computations, comparisons, and manipulations within a program.
Python supports several types of operators:

Arithmetic Operators in Python revolves around their functionality in performing various mathematical operations on numeric values. These operators abide by standard mathematical rules.

1.Addition (+)

  • Adds two operands together.
  • If both operands are numbers, it performs addition.
  • f one or both operands are strings, it concatenates them.

2.Subtraction (-)

  • Subtracts the right operand from the left operand.
  • If used with numeric values, it performs subtraction.

3.Multiplication (*)

  • Multiplies two operands.
  • When used with numbers, it performs multiplication.
  • If used with strings and a numeric value (e.g., 'a' * 3), it repeats the string the specified number of times.

4.Division (/)

  • Divides the left operand by the right operand.
  • Always returns a floating-point number, even if the division has no remainder.

5.Modulo (%)

  • Returns the remainder of the division of the left operand by the right operand.
  • Useful for determining divisibility or extracting remainders.

6.Exponential (**)

  • Raises the left operand to the power of the right operand.
  • Computes exponential values.

7.Floor Division (//)

  • Returns the quotient of the division, rounded down to the nearest whole number.
  • Ignores the decimal part of the result.

Assignment operators in Python are used to assign values to variables. They provide a shorthand way to perform an operation on a variable and assign the result back to the same variable. The basic assignment operator in Python is the equals sign =.

1.Basic Assignment (=)

    Assigns the value on the right-hand side to the variable on the left-hand side.
    Example: x = 5 assigns the value 5 to the variable x.

2.Compound Assignment Operators

    These operators combine arithmetic or bitwise operations with assignment.
    They perform an operation between the left-hand side variable and the right-hand side value, and then assign the result to the left-hand side variable.
    For example:
    += adds the right operand to the left operand and assigns the result to the left operand.
    -= subtracts the right operand from the left operand and assigns the result to the left operand.
    *= multiplies the left operand by the right operand and assigns the result to the left operand.
    /= divides the left operand by the right operand and assigns the result to the left operand.
    %= performs modulo and assigns the remainder to the left operand.
    //= performs floor division and assigns the quotient to the left operand.
    **= raises the left operand to the power of the right operand and assigns the result to the left operand.
    &=, |= perform bitwise AND and OR operations, respectively, and assign the result to the left operand.

Comparison Operators in Python are used to compare two values or operands. These operators evaluate the relationship between the operands and return a Boolean value (True or False) based on the comparison result.

Equal to (==)

    Checks if the left operand is equal to the right operand.
    Example: 5 == 5 evaluates to True.

Not equal to (!=)

    Checks if the left operand is not equal to the right operand.
    Example: 5 != 3 evaluates to True.

Greater than (>)

    Checks if the left operand is greater than the right operand.
    Example: 7 > 4 evaluates to True.

Less than (<)

    Checks if the left operand is less than the right operand.
    Example: 2 < 6 evaluates to True.

Greater than or equal to (>=)

    Checks if the left operand is greater than or equal to the right operand.
    Example: 10 >= 10 evaluates to True.

Less than or equal to (<=)

    Checks if the left operand is less than or equal to the right operand.
    Example: 3 <= 5 evaluates to True.

Logical Operators in Python are used to perform logical operations on Boolean values (True or False) or expressions. These operators allow you to combine multiple conditions and evaluate them to produce a single Boolean result.

1.Logical AND (and)

    Returns 'True' if both operands or expressions on either side of the operator are 'True'.
    If any operand is 'False', the result is 'False'.
    Example: 'True and True' evaluates to 'True', while 'True and False' evaluates to 'False'.

2.Logical OR (or)

    Returns 'True' if at least one of the operands or expressions on either side of the operator is 'True'.
    If both operands are 'False', the result is 'False'.
    Example: 'True or False' evaluates to 'True', while 'False or False' evaluates to 'False'.

3.Logical NOT (not)

    Reverses the logical state of its operand
    If the operand is 'True', not makes it 'False'. If the operand is 'False', 'not' makes it 'True'.
    Example:'not True' evaluates to 'False', while 'not False' evaluates to 'True'.

Identity Operators are used to compare the memory locations of two objects to determine if they are referencing the same object in memory. These operators are used with objects such as variables, lists, dictionaries, etc.

1.is Operator

    The 'is' operator checks if two operands refer to the same object in memory.
    It returns 'True' if the operands point to the same object, otherwise 'False.'
    Example:
      x = [1, 2, 3]
      y = [1, 2, 3]
      z = x

      print(x is y) # Output: False (Different memory locations)
      print(x is z) # Output: True (Both x and z point to the same object)

2.is not Operator

    The 'is not' operator checks if two operands do not refer to the same object in memory.
    It returns 'True' if the operands do not point to the same object, otherwise 'False'.
    Example:
      a = "Hello"
      b = "Hello"

      print(a is not b) # Output: True (Different memory locations even though the values are the same)

Membership Operators in Python are used to test whether a value or variable is present within a sequence or collection. These operators determine if a specified element belongs to a certain data structure like a string, list, tuple, set, or dictionary.

1.in Operator

    The 'in' operator checks if a value or variable exists in a sequence.
    It returns 'True' if the value is found in the sequence, otherwise 'False'.
    Example:
      my_list = [1, 2, 3, 4, 5]
      print(3 in my_list) # Output: True (3 is present in the list)
      print(6 in my_list) # Output: False (6 is not present in the list)

2.not in Operator

    The 'not in' operator checks if a value or variable does not exist in a sequence.
    It returns 'True' if the value is not found in the sequence, otherwise 'False'.
    Example:
      my_tuple = (10, 20, 30)
      print(15 not in my_tuple) # Output: True (15 is not present in the tuple)
      print(20 not in my_tuple) # Output: False (20 is present in the tuple)
