    Simple decorators with parameters

    Decorators by taking the parameters in the function


  • In the above example divide() function was taken and and assigned a, b parameters for performing the division operation.
  • Here, when we call the divide() function along with arguments, the inner() function present in division() decorator is called instead.
  • Then this inner() function calls the original divide() function with the arguments & returns the results.

    • Combining Decorators

    Multiple decorators can be linked together.For combining the decorators in Python, we can apply multiple decorators to a single function by placing them one after the other, with the most inner decorator being applied first.
