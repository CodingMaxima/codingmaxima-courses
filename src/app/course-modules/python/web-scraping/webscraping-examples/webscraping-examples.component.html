Web Scraping Examples

Examples
Scraping Data from a Single Web Page:
import requests
from bs4 import BeautifulSoup

# Send a GET request to the web page
response = requests.get('https://www.example.com')

# Create a BeautifulSoup object to parse the HTML content
soup = BeautifulSoup(response.text, 'html.parser')

# Extract specific data from the page
title = soup.title.text
paragraph = soup.find('p').text

# Print the extracted data
print('Title:', title)
print('Paragraph:', paragraph)
Scraping Data from Multiple Pages:
import requests
from bs4 import BeautifulSoup

# Send a GET request to the web page
response = requests.get('https://www.example.com')

# Create a BeautifulSoup object to parse the HTML content
soup = BeautifulSoup(response.text, 'html.parser')

# Find all links on the page
links = soup.find_all('a')

# Extract data from each link
for link in links:
    href = link['href']
    text = link.text

    # Print the extracted data
    print('Link:', href)
    print('Text:', text)
Scraping Data from a Table:
import requests
from bs4 import BeautifulSoup

# Send a GET request to the web page
response = requests.get('https://www.example.com/table')

# Create a BeautifulSoup object to parse the HTML content
soup = BeautifulSoup(response.text, 'html.parser')

# Find the table on the page
table = soup.find('table')

# Extract data from each row of the table
rows = table.find_all('tr')
for row in rows:
    cells = row.find_all('td')

    # Extract data from each cell
    for cell in cells:
        data = cell.text

        # Print the extracted data
        print('Data:', data)
Remember to install the required libraries ( `requests ` and `beautifulsoup4 `) using pip before running these examples. These examples demonstrate the basics of web scraping, but the techniques can be further expanded and customized based on the structure of the target website and the specific data you want to extract.
