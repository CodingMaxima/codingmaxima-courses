Single Inheritance Example

introduction Program

In this example, we have a parent class Calculator that represents a basic mathematical calculator. It includes methods for addition, subtraction, multiplication, and division. The child class ScientificCalculator extends Calculator to inherit the basic calculator functionality and adds additional methods for square root and exponentiation using the Math class. In the SingleInheritanceExample class, we create an instance of Calculator called calculator and use it to perform addition and subtraction operations. We also create an instance of ScientificCalculator called scientificCalculator and use it to calculate the square root and raise a number to a power.
