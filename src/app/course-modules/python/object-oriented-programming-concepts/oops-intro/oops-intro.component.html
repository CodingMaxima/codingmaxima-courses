Class

It is an collection of objects. It is a logical set-up which has specific methods and attributes.

Ex:- If a student class were there their must be some attributes and methods like name, id, email, marks, branch college_name, etc..

object

It is an set-up which has both state and behavior.It was like an real world set-ups.In python everything was an object. They have both attributes and methods.

EX:- peacock, It was an object.

It has state (color, age, name,species) and behavior (dancing, response, singing)

Encapsulation

It is one of the main concept in oops. It helps to wrap the data and methods into the single unit. It protects from the others classes to access the methods and attributes in the current class. We can't access the other functions data easily. so it is also called as data binding.

Ex:- In the student class, there are some attributes and methods like name , marks, id etc.. Once we make the attribute as private we can't modify the data into other class. Inheritance

It is defined as obtaining the properties and behaviour of the parent class to the child class.The main purpose of using the inheritance is for code re-usability I.e., we no need to write the code again and again can use from the already existing code.

Ex:- From a country class , we are creating the a city class. Then we need to mention common attributes and methods again and again in the city class just need to add those attributes and methods which are specific to the city.

Polymorphism

It is used to perform the single task in different ways.

Ex:- The Teacher acts as an teacher for students in class, acts as colleague/friend in staff room and father/ husband in the home.
