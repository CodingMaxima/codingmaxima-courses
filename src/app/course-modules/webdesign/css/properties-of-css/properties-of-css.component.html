  • CSS (Cascading Style Sheets) properties are used to style and format the appearance of HTML elements on a webpage.
  • They allow you to control various aspects of how elements are displayed, such as their colors, sizes, positions, borders, and more.

    • Here are some common types of CSS properties:

    1.Color and Background

Property Description Values
color Sets the color of a text RGB, hex, keyword
background-color Sets the background color of an element. hexadecimal value, rgb value, or as a named color.
background-image Sets one or multiple background images for an element. URL of the image location or named color.

    2.Fonts and Text

Property Description Values
font-family Specifies the font family for text family-name, generic-family, inherit
font-size Specifies the font size of text xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, length, %, inherit
font-weight Specifies the weight of a font normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900, inherit Careful, many of these are not supported!
text-align Aligns the text in an element left, right, center, justify
text-decoration Adds decoration to text none, underline, overline, line-through

    3.Box Model

Property Description Values
margin Sets the space outside an element. margin-top,margin-right,margin-bottom,margin-left
padding Sets the space inside an element. top, right, bottom, and/or left
border Defines the border properties (width, style, color) top, right, bottom, and/or left
width and height Sets the width and height of an element numeric value (like pixels, (r)em, percentages)

    4.Layout

Property Description Values
display Specifies how an element is displayed. block,inline-block,none,flex,grid
position Sets the positioning method of an element. static,relative,fixed,absolute,sticky
float Allows elements to float to the left or right. right,left,none

  • Advanced CSS properties go beyond the basics and allow you to create more complex and sophisticated styles for your web pages.

    • Here are some advanced CSS properties:

    1.Flexbox

Property Description Values
display-flex Enables a flex container. flex
flex-direction Specifies the direction of the main axis. row | row-reverse | column | column-reverse
justify-content Aligns items along the main axis. flex-start | flex-end | center | space-between | space-around | space-evenly
align-items Aligns items along the cross axis. flex-start | flex-end | center | baseline | stretch

    2.Grid Layout

Property Description Values
display-grid Enables a grid container. grid
grid-template Defines the number and size of rows and columns. grid-template-rows and grid-template-columns
grid-gap Specifies the gap between grid items. grid-gap or gap

    3.Box Border Model

Property Description Values
border A shorthand property that combines border-width, border-style, and border-color in one declaration. numeric value (like pixels, percentages), style name or as a named color.
border-radius Defines the radius of the border corners, creating rounded corners. numeric value (like pixels, percentages)
box-shadow While not strictly a border property, box-shadow can be used to create a shadow effect, simulating multiple borders or enhancing the appearance of the element.(width, color) numeric value (like pixels, percentages) or as a named color.
border-image Allows you to use an image as the border, rather than a simple color or style. Requiered image or as a named color.
box-shadow The box-shadow property in CSS is used to add a shadow effect to an element.
Syntax: box-shadow: h-offset v-offset blur spread color inset;		 h-offset(horizontal offset)
v-offset(vertical offset) blur(numeric value) spread(numeric value) color(named of the color or color code)

    4.Selectors

Property Description Values
box-hover the :hover pseudo-class is used to select and style an element when the user hovers over it with the mouse. named of the color or color code
nth-child(n) The :nth-child pseudo-class is used to select and style elements based on their position within a parent container.
Syntax: :nth-child(an+b);

n is the cycle size (e.g., every 2nd element, every 3rd element).

a is the multiplier.

b is the offset or starting point.

 even , odd or number value
