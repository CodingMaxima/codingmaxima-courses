Introduction code and output

In HTML, lists are used to organize and display content in a structured manner.

They allow you to group related items together, making information more organized and easily digestible for website visitors.

There are different types of HTML Lists:

1.Unordered Lists (<ul>):

  • Unordered lists are used to present items in a bulleted format without any specific sequence or order.
  • Each item within an unordered list is marked with a bullet point by default.
  • They are created using the <ul> tag, and individual list items are denoted by<li> tags.
Example

<ul>

    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>

</ul>

2.Ordered Lists (<ol>):

  • Ordered lists are used to present items in a numbered sequence.
  • Each item in an ordered list is automatically numbered by default (1, 2, 3...).
  • They are created using the <ol> tag, and like unordered lists, individual list items are represented by <li> tags.
Example

<ol>

    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>

</ol>

Nested lists(<ol>):

  • Nested lists in HTML refer to the concept of placing one list inside another list.
  • This allows for the creation of hierarchical structures, where a list item within a parent list can contain its own sublist.
  • You can nest any type of list (unordered, ordered, or definition lists) within another list item.
Example

<ul>

    <li>Mani Item-1</li>
    <li>Mani Item-2

    <ul>

      <li>Sub Item-1</li>
      <li>Sub Item-2</li>

    </ul>
    </li>

    <li>Mani Item-3</li>

</ul>

Description list(<dl>):

  • In HTML, the description list (<dl>) is a type of list used to display terms and their corresponding definitions or descriptions in a structured format.
  • It's comprised of pairs of terms and their associated explanations or definitions.
  • The <dl> element represents the description list container, and it consists of two types of child elements:
    • <dt> (Definition Term): This tag defines the term being described or explained.
    • <dd> (Definition Description): This tag contains the description or definition of the term specified within the <dt> tag.
Example

<dl>

    <dt>HTML</li>
    <dd>HyperText Markup Language - a markup language used for creating web pages.</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets - a style sheet language used for describing the presentation of a document written in HTML.</dd>

</dl>
