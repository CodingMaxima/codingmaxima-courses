The <video> tag in HTML is used to embed video content directly into a web page.

It allows you to specify various attributes and settings to control the appearance and behavior of the video.

A basic example of how the <video> tag is used:

Example

<video width="640" height="360" controls>

    <source src="video.mp4" type="video/mp4">

</video>

Explanation of the attributes used:

  • width: Specifies the width of the video player in pixels.
  • height: Specifies the height of the video player in pixels.
  • controls: Adds default video controls (such as play, pause, volume control, etc.) to the video player.
  • <source>: This is a child element of the <video> tag used to specify the video file to be displayed.
    It allows multiple <source> elements to provide different file formats to ensure compatibility across various browsers.
    In the example above, video.mp4 is the video file in MP4 format, and type="video/mp4" indicates the MIME type of the video file.

The <img> tag in HTML is used to display an image on a web page.

It is a self-closing tag and does not have a closing tag.

A basic example of how the <img> tag is used:

Structure

<img src="image-url" alt="alternate-text" width="width-value" height="height-value">

Explanation of the attributes used in the <img> tag:

  • src: Specifies the URL or path to the image file.
  • alt: Provides alternative text for the image, which is displayed if the image cannot be loaded or for accessibility purposes (screen readers for visually impaired users).
  • width: Specifies the width of the image in pixels (optional attribute).
  • height: Specifies the height of the image in pixels (optional attribute).
  • <source>: This is a child element of the <video> tag used to specify the video file to be displayed.
    It allows multiple <source> elements to provide different file formats to ensure compatibility across various browsers.
    In the example above, video.mp4 is the video file in MP4 format, and type="video/mp4" indicates the MIME type of the video file.
Example

<img src="example.jpg" alt="Example Image" width="300" height="200">

The <audio> tag in HTML is used to embed sound content in a web page.

A basic example of how the <audio> tag is used:

Example

<audio controls>

    <source src="audiofile.mp3" type="audio/mpeg">

</audio>

Explanation of the attributes used:

  • controls: This attribute displays the audio player controls (such as play, pause, volume control, etc.).
  • <source>: This tag specifies the audio file to be played. It allows you to specify multiple sources using different formats to ensure compatibility with various browsers.
    In the example above, src points to the audio file (audiofile.mp3) and type specifies the MIME type of the audio file.
