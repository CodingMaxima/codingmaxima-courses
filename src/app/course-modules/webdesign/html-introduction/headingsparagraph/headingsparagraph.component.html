In HTML (Hypertext Markup Language), a paragraph is a structural element used to define and display text content as a separate block.

It is represented by the <p> tag.

The <p> tag is an opening tag that precedes the text content and a closing tag </p> that follows the text content.

You can have multiple <p> tags in a single HTML document to create several paragraphs of text, each encapsulated within its own <p> opening and closing tags.

Example

<p>This is the first paragraph.</p>

<p>This is another paragraph.</p>
