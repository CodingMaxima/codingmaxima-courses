intersection of two arrays

Given two arrays a[] and b[] respectively of size n and m, the task is to print the count of elements in the intersection (or common elements) of the two arrays. For this question, the intersection of two arrays can be defined as the set containing distinct common elements between the two arrays.

Example 1

Input: n = 5, m = 3
A[] = {{ '{' }}89, 24, 75, 11, 23{{ '}' }}
B[]={{ '{' }}89, 2, 4{{ '}' }}

Output: 1

Explanation: 89 is the only element in the intersection of two arrays.

Example 2

Input: n = 6, m = 5,
A[] = {{ '{' }}1, 2, 3, 4, 5, 6{{ '}' }}
B[]={{ '{' }}3, 4, 5, 6, 7{{ '}' }}

Output: 4

3 4 5 and 6 are the elements in the intersection of two arrays
