Product array puzzle(8 times asked ) Given an array nums[] of size n, construct a Product Array P (of same size n) such that P[i] is equal to the product of all the elements of nums except nums[i].

Example 1

Input: N = 5, Arr[] = {{ '{' }}10, 3, 5, 6, 2{{ '}' }}

Output: -180 600 360 300 900

Explanation: For i=0, P[i] = 3*5*6*2 = 180. For i=1, P[i] = 10*5*6*2 = 600. For i=2, P[i] = 10*3*6*2 = 360. For i=3, P[i] = 10*3*5*2 = 300. For i=4, P[i] = 10*3*5*6 = 900.

Example 2

Input: N = 2, Arr[] = {{ '{' }}12,0{{ '}' }}

Output: 0 12

Explanation: Since, 3 is present more than N/2 times, so it is the majority element
