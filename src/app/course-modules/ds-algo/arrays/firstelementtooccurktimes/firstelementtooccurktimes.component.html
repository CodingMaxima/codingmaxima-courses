First element to occurs k times

Problem Statement: Given an array of N integers, you are required to find the first element that occurs at least K number of times in the array. If no such element exists, return a special value or indicate that there is no such element

Example 1

Input: Input: Array: [3, 1, 4, 2, 2, 1, 5, 3]
K: 2

Output: 2

In this example, the element "2" is the first element that occurs at least 2 times in the given array.

Example 2

Output: No element occurring at least 3 times found.
