Kadane's Algorithm

Kadane's algorithm is a brilliant technique in computer science used to find the maximum sum subarray within a given array of numbers. This efficient algorithm scans through the array in a single pass, dynamically tracking the maximum subarray sum ending at each position. By cleverly considering the local and global maximums, Kadane's algorithm elegantly solves the 'maximum subarray problem' with a time complexity of O(n), making it a go-to solution for optimizing a wide range of applications, from financial analysis to image processing.

Example 1

Input: N = 5, Arr[] = {{ '{' }}1, 2, 3, -2, 5{{ '}' }}

Output: 9

Explanation: The maximum subarray sum is 9, which corresponds to the elements (1, 2, 3, -2, 5) forming a contiguous subarray.

Example 2

Input: N = 4, Arr[] = {{ '{' }}-1, -2, -3, -4{{ '}' }}

Output: -1

Explanation: The maximum subarray sum is -1, which is obtained from the element -1 itself.
