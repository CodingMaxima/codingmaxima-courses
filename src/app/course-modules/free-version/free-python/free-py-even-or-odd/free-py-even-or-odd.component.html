Even or Odd

Problem Statement Program Animation Video Test Cases Practice

An even number is any integer (whole number) that is exactly divisible by 2, meaning that when the number is divided by 2, the remainder is 0. Examples of even numbers include 2, 4, 6, 8, 10, 12, etc.

On the other hand, an odd number is any integer that is not divisible by 2, meaning that when the number is divided by 2, the remainder is always 1. Examples of odd numbers include 1, 3, 5, 7, 9, 11, etc.

One way to quickly determine if a number is even or odd is to look at the last digit of the number. If the last digit is 0, 2, 4, 6, or 8, the number is even. If the last digit is 1, 3, 5, 7, or 9, the number is odd.

Example - 1
Input

Enter a number

22

Output

22 is even number.

Example - 2
Input

Enter a number

11

Output

11 is odd number.

Example - 3
Input

Enter a number

85

Output

85 is odd number.
