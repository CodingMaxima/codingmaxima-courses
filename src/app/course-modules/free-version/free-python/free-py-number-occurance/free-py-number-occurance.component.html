Number Occurance(Dictionaries)

Problem Statement Program Animation Video Test Cases Practice

The program is to print the how many times each number was repeated in the list by using the dictionary.

Suppose taken an list of values list = [1, 2, 3, 3, 2, 4, 2]. From the above list have to find the each value how many times repeated.

The result will be like 1- 1 time , 2- 3 times, 3 - 2 times, 4- 1time

Example - 1
Input

numbers = [9, 18, 23, 16, 5, 18, 23, 5, 5]

Output

{{'{'}}9: 1, 18: 2, 23: 2, 16: 1, 5: 3{{'}'}}

Example - 2
Input

numbers = [1, 1, 1, 5, 5, 3, 1, 3, 3, 1, 4, 4, 4, 2, 2, 2, 2]

Output

{{'{'}}1: 5, 5: 2, 3: 3, 4: 3, 2: 4{{'}'}}

Example - 3
Input

numbers = [23, 54, 87, 23, 54, 98, 23]

sum = 23

Output

{{'{'}}23: 3, 54: 2, 87: 1, 98: 1{{'}'}}
