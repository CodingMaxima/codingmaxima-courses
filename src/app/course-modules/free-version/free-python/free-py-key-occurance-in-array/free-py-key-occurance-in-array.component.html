Key Count in List

Problem Statement Program Animation Video Test Cases Practice

The program is to print how many times the value which we are given as an input is present in the list of values

Suppose if we are giving the input list as list = [1, 2, 34, 55, 34, 12, 22]

from the list we are taken one value(key value) i.e., 34 .so from the list we have to find how many times the key value present in the array that means it will check the value 34 in the list it is present so the output will be 2 times found in the list .

If we giving 5 as key value it should have to print 0 times the key value present in the list because there is no such value in the list.

Example - 1
Input

list = [12, 64, 64, 17, 82, 82, 91, 72, 17, 82]

Key = 82

Output

The 82 occured in the array is 3 times

Example - 2
Input

list = [11, 13, 85, 22, 60, 66, 34]

Key = 55

Output

The 55 occured in the array is 0 times

Example - 3
Input

list = [24, 22, 85, 22, 22]

Key = 22

Output

The 22 occured in the array is 3 times
