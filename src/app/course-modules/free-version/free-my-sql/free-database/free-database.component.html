The CREATE DATABASE statement is used to create a new SQL(Structure Query Language) database. 


                        Syntax : CREATE DATABASE databasename;
                                        
                        E.g.: - CREATE DATABASE myDatabase;
Example - 1
Query

CREATE DATABASE myDatabase;

Result of Query

DATABASE CREATED.

Example - 2
Query

CREATE DATABASE bookingDatabase;

Result of Query

DATABASE CREATED.


Introduction queries Animation video

When you have multiple databases in your SQL Schema, then before starting your operation, you would need to select a database where all the operations would be performed. The SQL USE statement is used to select any existing database in the SQL schema. 


                            Syntax : USE databasename;
                                            
                            E.g.: - USE myDatabase;
Example - 1
Query

USE myDatabase;

Result of Query

DATABASE USED.

Example - 2
Query

USE bookingDatabase;

Result of Query

DATABASE USED.


Query

SHOW DATABASES;

USE myDatabase;

The SQL DROP DATABASE statement is used to drop an existing database in SQL schema. 


                            Syntax : DROP DATABASE databaseName;
                                            
                            E.g.: - DROP DATABASE myDatabase;
Example - 1
Query

DROP DATABASE myDatabase;

Result of Query

DATABASE droped.

Example - 2
Query

DROP DATABASE bookingDatabase;

Result of Query

DATABASE droped.

Query

SHOW DATABASES;

DROP DATABASE myDatabase;

SHOW DATABASES;
