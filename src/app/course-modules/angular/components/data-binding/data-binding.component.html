Data Binding Introduction

1.Interpolation (` {{'{'}}{{'{'}} {{'}'}}{{'}'}} `):

Interpolation is a one-way binding method that allows you to embed expressions in the text content of HTML elements or attributes.

These expressions are evaluated and replaced with their corresponding values. It's commonly used to display dynamic data in the UI.

Example: 


<h1>Welcome, {{'{'}}{{'{'}} username {{'}'}}{{'}'}}!</h1>

In this example, the `username` variable's value will be dynamically displayed in the <h1> element.

2.Property binding ( `[ ]` ):

Property binding allows you to bind the value of a component property to an HTML element's property.

This enables you to update the property's value in the component, and the change will automatically reflect in the UI.

Example: 


<input [value]="message">

Here, the `value` property of the <input> element is bound to the message property of the component. Any changes made to the message property will be reflected in the input field.

3.Event binding ( `( )` ):

Event binding enables you to respond to user actions, such as button clicks or keystrokes, by binding the component's methods to the corresponding event in the HTML template. When the event occurs, the bound method is executed.

Example: 


<button (click)="handleClick()">Click me</button>

In this case, the `handleClick()` method of the component will be invoked when the user clicks the button.

4.Two-way binding ( `[( )]` ):

Property binding allows you to bind the value of a component property to an HTML element's property.

Two-way binding combines both property binding and event binding, allowing you to establish a two-way communication between the component and the UI. It ensures that any changes made to the property in the component or the input value in the UI are automatically synchronized

Example: 


<input [(ngModel)]="username">

Here, the `ngModel` directive is used for two-way binding. The `username` property of the component will be updated with the input value, and any changes to the `username` property will update the input value.
