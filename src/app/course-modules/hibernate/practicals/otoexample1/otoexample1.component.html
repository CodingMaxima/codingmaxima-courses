Employee Class

create a Employee Class by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. emp_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. emp_name - it should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. salary - it should be in between greater than or equal to 5000 and less than or equal to 100000(hint: use "CHECK CONSTRAINT")
  4. emp_joining_date - keep a fixed value for it. If you doesn't want to give an value for this particular column/for auto generating of the value without giving any manual values(hint: use "DEFAULT CONSTRAINT")
Laptop Class

create a Laptop Class by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. lap_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. lap_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. lap_model - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  4. lap_warranty - It is warranty end date. It should not be empty (hint: use "NOT NULL CONSTRAINT")
  5. emp_id - It is connection to the employee table emp_id. (hint: use "FOREIGN KEY")
