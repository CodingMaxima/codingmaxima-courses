Apartment Class

create an constraints table of Apartment by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. apartment_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. apartment_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. no_flats - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  4. facing - It should be either 'east','west','north','south'(hint: use "CHECK CONSTRAINT")
  5. no_floors - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  6. isLift - It should be either present or not present & It should not be empty (hint: use "NOT NULL CONSTRAINT")
Flat Class

create an constraints table of Flat by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. flat_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. owner_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. no_bedrooms - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  4. facing - It should be either 'east','west','north','south'(hint: use "CHECK CONSTRAINT")
  5. floor - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  6. isAvailable - It should be either present or not present & It should not be empty (hint: use "NOT NULL CONSTRAINT")
  7. apartment_id - It is connection to the Apartment table apartment_id. (hint: use "FOREIGN KEY")
