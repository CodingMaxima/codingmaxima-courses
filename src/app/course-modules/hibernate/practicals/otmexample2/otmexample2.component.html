Employee Class

create an constraints table of employee by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. emp_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. emp_name - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  3. gender - It should not be empty (hint: use "NOT NULL CONSTRAINT")
  4. salary - It should be not less than 10000 and not more than 120000(hint: use "CHECK CONSTRAINT")
  5. experience - It should be minimum zero
  6. des_id - It is connection to the designation table des_id. (hint: use "FOREIGN KEY")
Designation Class

create an constraints table of designation by using below data

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.
  1. des_id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT")
  2. des_name - It should not be empty (hint: use "NOT NULL ,UNIQUE CONSTRAINTs")
  3. department - They should be either 'Technical','Non Technical', 'Finance', 'Support' any one of those(hint: use "CHECK CONSTRAINT")
