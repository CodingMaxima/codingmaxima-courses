Hibernate Configuration

Introduction
Property Description
hibernate.dialect This property makes Hibernate generate the appropriate SQL for the chosen database.
hibernate.connection.driver_class It represents the JDBC driver class.
hibernate.connection.url It represents the JDBC URL.
hibernate.connection.username It represents the database username.
hibernate.connection.password It represents the database password.
hibernate.hbm2ddl.auto
  • It automatically generates a schema in the database with the creation of SessionFactory.
  • If you want to create a new schema then
  • hibernate.hbm2ddl.auto = create.
  • If you want to update schema then
  • hibernate.hbm2ddl.auto = update.

    hibernate.show_sql
  • It is used to display the executed SQL statements to console.
  • If you want to show sql commands then
  • hibernate.show_sql = true.
  • By default it is false.
