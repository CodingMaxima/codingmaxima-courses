Employee- Class
  • Id - make sure it should not be duplicate and should not be empty (hint: use "PRIMARY KEY CONSTRAINT").
  • name - It should not be empty (hint: use "NOT NULL CONSTRAINT").
  • emailId - it should not be duplicate (hint: use "Unique CONSTRAINT")
  • designation - It should not be empty (hint: use "NOT NULL CONSTRAINT").
  • salary- t should be in between greater than or equal to 12000 and less than or equal to 100000(hint: use "CHECK CONSTRAINT")
  • joining_date - keep a fixed value for it. If you doesn't want to give an value for this particular column/for auto generating of the value without giving any manual values(hint: use "DEFAULT CONSTRAINT")
  • With constructors, all getters and setters and toString methods
